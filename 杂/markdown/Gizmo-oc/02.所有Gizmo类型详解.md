# Blender 所有 Gizmo 类型详解

## 1. 概述

Blender 的 Gizmo 系统提供了多种预定义的 Gizmo 类型，用于实现 3D 视图中的交互控件。所有内置 Gizmo 类型在 `gizmo_library/gizmo_types/` 目录中定义。

### 内置 Gizmo 类型列表

| Gizmo 类型 | 说明 | 文件位置 |
|-----------|------|----------|
| GIZMO_GT_arrow_3d | 3D 箭头 gizmo | gizmo_library/gizmo_types/arrow3d_gizmo.cc:534-597 |
| GIZMO_GT_button_2d | 2D 按钮 gizmo | gizmo_library/gizmo_types/button2d_gizmo.cc:400-453 |
| GIZMO_GT_cage_2d | 2D 笼状 gizmo | gizmo_library/gizmo_types/cage2d_gizmo.cc:1412-1474 |
| GIZMO_GT_cage_3d | 3D 笼状 gizmo | gizmo_library/gizmo_types/cage3d_gizmo.cc:635-693 |
| GIZMO_GT_dial_3d | 3D 刻度盘 gizmo | gizmo_library/gizmo_types/dial3d_gizmo.cc:670-735 |
| GIZMO_GT_move_3d | 3D 移动 gizmo | gizmo_library/gizmo_types/move3d_gizmo.cc:435-479 |
| GIZMO_GT_primitive_3d | 3D 原始形状 gizmo | gizmo_library/gizmo_types/primitive3d_gizmo.cc:253-309 |
| GIZMO_GT_blank_3d | 空白 gizmo（仅事件捕获） | gizmo_library/gizmo_types/blank3d_gizmo.cc:41-59 |

## 2. 核心内置 Gizmo 类型详解

### 2.1 <span style="color:#E91E63">GIZMO_GT_arrow_3d</span>

#### 定义位置
**实现文件**: `source/blender/editors/gizmo_library/gizmo_types/arrow3d_gizmo.cc:534-597`

#### 功能
<span style="color:#E91E63">3D 箭头 gizmo</span>，用于沿指定方向进行拖动交互。箭头可以有不同的头部形状。

#### 支持的样式（`ED_gizmo_library.hh:54-60`）
```cpp
enum {
  ED_GIZMO_ARROW_STYLE_NORMAL = 0,    // 标准锥形箭头
  ED_GIZMO_ARROW_STYLE_CROSS = 1,     // 十字交叉箭头
  ED_GIZMO_ARROW_STYLE_BOX = 2,       // 方块箭头
  ED_GIZMO_ARROW_STYLE_CONE = 3,      // 圆锥箭头
  ED_GIZMO_ARROW_STYLE_PLANE = 4,     // 平面箭头
};
```

#### Transform Flags（`ED_gizmo_library.hh:62-68`）
```cpp
enum {
  ED_GIZMO_ARROW_XFORM_FLAG_INVERTED = (1 << 3),   // 交互时反转偏移
  ED_GIZMO_ARROW_XFORM_FLAG_CONSTRAINED = (1 << 4), // 限制箭头交互到属性宽度
};
```

#### Draw Flags（`ED_gizmo_library.hh:70-75`）
```cpp
enum {
  ED_GIZMO_ARROW_DRAW_FLAG_STEM = (1 << 0),    // 显示箭头杆
  ED_GIZMO_ARROW_DRAW_FLAG_ORIGIN = (1 << 1), // 显示原点
};
```

#### 关键属性
| 属性 | 类型 | 说明 |
|--------|------|------|
| `offset` | float | 箭头偏移量 |
| `length` | float | 箭头长度（`arrow3d_gizmo.cc:587`） |
| `aspect` | float[2] | 锥形/方块样式的长宽比（`arrow3d_gizmo.cc:588-589`） |

#### 使用示例
- **view3d_gizmo_camera.cc**: 相机距离控制
- **view3d_gizmo_light.cc**: 灯光参数调节
- **transform_gizmo_3d.cc**: 变换操作轴指示

#### 绘制实现（`arrow3d_gizmo.cc:78-223`）
`arrow_draw_geom()` 函数处理不同样式的箭头绘制：
- `NORMAL`: 标准圆锥 + 圆柱形箭头杆
- `CROSS`: 十字交叉线（`arrow3d_gizmo.cc:95-104`）
- `BOX`: 立方体箭头头（`arrow3d_gizmo.cc:171-183`）
- `CONE`: 圆锥形箭头头（`arrow3d_gizmo.cc:106-120`）
- `PLANE`: 平面形箭头（`arrow3d_gizmo.cc:122-145`）

## 5. 总结

### 5.1 Gizmo 架构层次

```
wmGizmoType (Gizmo 类型定义)
    ├── ID 名称
    ├── 回调函数
    │   ├── draw() / draw_select()
    │   ├── test_select()
    │   ├── invoke()
    │   ├── modal()
    │   └── exit()
    ├── struct_size (结构体大小)
    └── RNA 属性定义

wmGizmo (Gizmo 实例)
    ├── state (高亮/模态状态)
    ├── flag (绘制标志)
    ├── matrix_basis (基础矩阵)
    ├── matrix_offset (偏移矩阵)
    ├── matrix_space (空间矩阵)
    └── color (颜色)
```

### 5.2 最佳实践

1. **选择合适的 Gizmo 类型**
   - 单方向操作：使用 `GIZMO_GT_arrow_3d`
   - 旋转操作：使用 `GIZMO_GT_dial_3d`
   - 多操作变换：使用 `GIZMO_GT_cage_*`
   - 简单移动：使用 `GIZMO_GT_move_3d`

2. **属性绑定**
   - 使用 `WM_gizmo_target_property_def()` 定义目标属性
   - 使用 `WM_gizmo_target_property_def_rna()` 绑定 RNA 属性

3. **样式定制**
   - 设置适当的 `draw_style` 和 `draw_options`
   - 使用 `color` 属性自定义颜色
   - 通过 `matrix_basis` 控制位置和方向

4. **交互优化**
   - 实现 `test_select()` 提供精确的点击区域
   - 在 `modal()` 中处理 `WM_GIZMO_TWEAK_PRECISE` 精确模式
   - 支持吸附功能提升用户体验
