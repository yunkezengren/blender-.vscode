## 1. 概述

Gizmo 的属性绑定系统是连接交互工具与实际数据的桥梁。Blender 提供了两种绑定方式：

- **RNA 绑定**: 直接绑定到 Blender 数据结构的属性
- **自定义函数绑定**: 通过回调函数处理复杂的数据转换

<mark style="background-color: #2E7D32; color: white; padding: 2px 6px; border-radius: 3px;">★ Insight</mark>
属性绑定的核心是**解耦**：gizmo 不需要知道数据的具体存储方式，只需要通过统一的接口读写数据。

## 2. 属性绑定基础

### 2.1. wmGizmoProperty 结构

**源码位置**: `source/blender/windowmanager/gizmo/WM_gizmo_types.hh:232-260`

```cpp
struct wmGizmoProperty {
  wmGizmoProperty *next, *prev;

  char idname[64];                      // 属性名称（如 "offset", "matrix"）

  /* RNA 绑定 */
  PointerRNA ptr;                       // RNA 数据指针
  PropertyRNA *prop;                    // RNA 属性
  int index;                            // 数组索引（-1 表示非数组）

  /* 自定义函数绑定 */
  struct {
    wmGizmoPropertyFnGet value_get_fn;  // 获取函数
    wmGizmoPropertyFnSet value_set_fn;  // 设置函数
    wmGizmoPropertyFnRangeGet range_get_fn; // 范围函数
    void *user_data;                    // 用户数据
  } custom_func;

  /* 函数指针 - 指向实际的获取/设置实现 */
  wmGizmoPropertyFnGet value_get_fn;
  wmGizmoPropertyFnSet value_set_fn;

  /* 类型信息 */
  wmGizmoPropertyType *type;            // 属性类型定义
};
```

**关键字段**:
- `idname`: 用于在 gizmo 中查找特定属性
- `ptr` + `prop`: RNA 绑定的核心
- `custom_func`: 自定义函数绑定
- `value_get_fn/set_fn`: 实际执行的函数指针

## 3. RNA 属性绑定

### 3.1. 绑定方法

**源码位置**: `source/blender/windowmanager/gizmo/intern/wm_gizmo_property.cc:150-200`

```cpp
void WM_gizmo_target_property_def_rna(wmGizmo *gz,
                                      const char *prop_name,
                                      PointerRNA *ptr,
                                      const char *rna_path,
                                      int index)
{
  wmGizmoProperty *gz_prop = MEM_callocN<wmGizmoProperty>(__func__);

  /* 1. 设置属性名称 */
  STRNCPY(gz_prop->idname, prop_name);

  /* 2. 保存 RNA 指针 */
  gz_prop->ptr = *ptr;
  gz_prop->prop = RNA_struct_find_property(ptr, rna_path);

  if (!gz_prop->prop) {
    /* 错误处理 */
    MEM_freeN(gz_prop);
    return;
  }

  gz_prop->index = index;

  /* 3. 设置函数指针 */
  gz_prop->value_get_fn = wm_gizmo_property_rna_get;
  gz_prop->value_set_fn = wm_gizmo_property_rna_set;

  /* 4. 查找类型 */
  gz_prop->type = WM_gizmotype_target_property_find(gz->type, prop_name);

  /* 5. 添加到 gizmo */
  BLI_addtail(&gz->properties, gz_prop);
}
```

**使用示例**:
```cpp
/* 绑定到对象位置 */
PointerRNA ptr = RNA_pointer_create_discrete(&ob->id, &RNA_Object, ob);
WM_gizmo_target_property_def_rna(gz, "offset", &ptr, "location", -1);

/* 绑定到矩阵 (合成器背景) */
PointerRNA ptr = RNA_pointer_create_discrete(&snode->id, &RNA_SpaceNode, snode);
WM_gizmo_target_property_def_rna(gz, "matrix", &ptr, "zoom", -1);
```

### 4.2. Getter 实现

**示例: 合成器裁剪节点**:
```cpp
static void gizmo_node_crop_prop_matrix_get(const wmGizmo *gz,
                                            wmGizmoProperty *gz_prop,
                                            void *value_p)
{
  float (*matrix)[4] = (float (*)[4])value_p;

  /* 1. 获取用户数据 */
  NodeBBoxWidgetGroup *crop_group = (NodeBBoxWidgetGroup *)gz->parent_gzgroup->customdata;
  const float2 dims = crop_group->state.dims;
  const float2 offset = crop_group->state.offset;
  bNode *node = (bNode *)gz_prop->custom_func.user_data;

  /* 2. 从节点读取输入 */
  rctf rct;
  node_input_to_rect(node, dims, offset, &rct);

  /* 3. 转换为矩阵格式 */
  matrix[0][0] = fabsf(BLI_rctf_size_x(&rct));  // 宽度
  matrix[1][1] = fabsf(BLI_rctf_size_y(&rct));  // 高度
  matrix[3][0] = (BLI_rctf_cent_x(&rct) - 0.5f) * dims[0];  // 中心 X
  matrix[3][1] = (BLI_rctf_cent_y(&rct) - 0.5f) * dims[1];  // 中心 Y
}
```

**关键点**:
- 从 `gz_prop->custom_func.user_data` 获取上下文
- 读取节点输入值
- 转换为 gizmo 需要的矩阵格式

### 4.3. Setter 实现

**示例: 合成器裁剪节点**:
```cpp
static void gizmo_node_crop_prop_matrix_set(const wmGizmo *gz,
                                            wmGizmoProperty *gz_prop,
                                            const void *value_p)
{
  const float (*matrix)[4] = (const float (*)[4])value_p;

  /* 1. 获取上下文 */
  NodeBBoxWidgetGroup *crop_group = (NodeBBoxWidgetGroup *)gz->parent_gzgroup->customdata;
  const float2 dims = crop_group->state.dims;
  const float2 offset = crop_group->state.offset;
  bNode *node = (bNode *)gz_prop->custom_func.user_data;

  /* 2. 从矩阵提取矩形 */
  rctf rct;
  node_input_to_rect(node, dims, offset, &rct);

  /* 3. 应用 gizmo 的变换 */
  BLI_rctf_resize(&rct, fabsf(matrix[0][0]), fabsf(matrix[1][1]));
  BLI_rctf_recenter(&rct,
                    ((matrix[3][0]) / dims[0]) + 0.5f,
                    ((matrix[3][1]) / dims[1]) + 0.5f);

  /* 4. 裁剪到有效范围 */
  rctf rct_isect{};
  rct_isect.xmin = offset.x / dims.x;
  rct_isect.xmax = offset.x / dims.x + 1;
  rct_isect.ymin = offset.y / dims.y;
  rct_isect.ymax = offset.y / dims.y + 1;
  BLI_rctf_isect(&rct_isect, &rct, &rct);

  /* 5. 写回节点输入 */
  node_input_from_rect(node, &rct, dims, offset);

  /* 6. 触发更新 */
  gizmo_node_bbox_update(crop_group);
}
```

**绑定过程**:

## 5. 双向绑定机制

### 5.1. Gizmo → 外部数据

**流程**:
```
1. 用户拖动 gizmo
2. gizmo.modal() 更新 matrix_offset
3. 调用 WM_gizmo_target_property_float_set_array()
4. 执行 value_set_fn
   ├─ RNA 绑定: RNA_property_float_set_array()
   └─ 自定义函数: 用户的 setter 实现
5. 数据写入外部
6. 触发更新 (RNA_property_update 或自定义)
```

**源码位置**: `source/blender/editors/gizmo_library/gizmo_library_intern.hh:500-550`

```cpp
static wmOperatorStatus gizmo_cage2d_modal(/*...*/)
{
  /* ... 计算变换 ... */

  /* 更新属性 */
  wmGizmoProperty *gz_prop = WM_gizmo_target_property_find(gz, "matrix");
  if (gz_prop->type != nullptr) {
    WM_gizmo_target_property_float_set_array(C, gz, gz_prop, &gz->matrix_offset[0][0]);
  }

  return OPERATOR_RUNNING_MODAL;
}
```

### 6.2. 裁剪 (自定义函数)

**完整流程**:

1. **节点输入** (归一化坐标):
   ```
   X: 0.2, Y: 0.3, Width: 0.5, Height: 0.4
   ```

2. **Getter 转换** → **Gizmo 矩阵**:
   ```
   matrix[0][0] = 0.5 * dims.x      // 宽度
   matrix[1][1] = 0.4 * dims.y      // 高度
   matrix[3][0] = (0.2 + 0.25 - 0.5) * dims.x  // 中心 X
   matrix[3][1] = (0.3 + 0.2 - 0.5) * dims.y   // 中心 Y
   ```

3. **用户拖动 gizmo**

4. **Setter 转换** → **节点输入**:
   ```
   从 matrix 提取 → 写入 X/Y/Width/Height
   ```

## 7. 属性更新流程

### 7.1. Gizmo 修改 → 外部数据

**完整调用栈**:
```
用户拖动
  ↓
gizmo_cage2d_modal()
  ↓
计算新矩阵
  ↓
WM_gizmo_target_property_float_set_array(C, gz, gz_prop, matrix)
  ↓
gz_prop->value_set_fn(gz, gz_prop, matrix)
  ↓
wm_gizmo_property_rna_set() 或自定义 setter
  ↓
RNA_property_float_set_array(ptr, prop, matrix)
  ↓
RNA_property_update(ptr.data, prop)
  ↓
外部数据更新
```

```cpp
struct RectTransformInteraction {

static wmOperatorStatus gizmo_cage2d_invoke(bContext *C, wmGizmo *gz, const wmEvent *event)
{
  RectTransformInteraction *data = MEM_callocN<RectTransformInteraction>(__func__);

  /* 保存原始状态 */
  copy_m4_m4(data->orig_matrix_offset, gz->matrix_offset);

  /* 保存到撤销栈 */
  WM_gizmo_set_flag(gz, WM_GIZMO_NEEDS_UNDO, true);

  gz->interaction_data = data;
  return OPERATOR_RUNNING_MODAL;
}
```

```cpp
static void gizmo_cage2d_exit(bContext *C, wmGizmo *gz, const bool cancel)
{
  RectTransformInteraction *data = (RectTransformInteraction *)gz->interaction_data;

  if (cancel) {
    /* 恢复原始值 */
    wmGizmoProperty *gz_prop = WM_gizmo_target_property_find(gz, "matrix");
    if (gz_prop->type != nullptr) {
      WM_gizmo_target_property_float_set_array(C, gz, gz_prop, &data->orig_matrix_offset[0][0]);
    }
    copy_m4_m4(gz->matrix_offset, data->orig_matrix_offset);
  }
  else {
    /* 确认：自动关键帧 */
    wmGizmoProperty *gz_prop = WM_gizmo_target_property_find(gz, "matrix");
    if (WM_gizmo_target_property_is_valid(gz_prop)) {
      WM_gizmo_target_property_anim_autokey(C, gz, gz_prop);
    }
  }

  if (data) {
    MEM_freeN(data);
  }
}
```

## 10. 总结

### 10.1. 绑定方式对比

| 方式 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| **RNA 绑定** | 简单、自动、类型安全 | 仅限 RNA 属性 | 直接属性（位置、缩放） |
| **自定义函数** | 灵活、可转换、可验证 | 需要手动实现 | 复杂转换、节点输入 |

### 10.2. 关键函数

| 函数 | 作用 |
|------|------|
| `WM_gizmo_target_property_def_rna()` | RNA 绑定 |
| `WM_gizmo_target_property_def_func()` | 自定义函数绑定 |
| `WM_gizmo_target_property_float_get_array()` | 读取值 |
| `WM_gizmo_target_property_float_set_array()` | 写入值 |
| `WM_gizmo_target_property_update()` | 外部→Gizmo |

### 10.3. 最佳实践

1. **选择合适的绑定方式**
   - 简单属性 → RNA
   - 复杂转换 → 自定义函数

2. **处理边界情况**
   - 检查指针有效性
   - 验证数据范围
   - 处理错误情况

3. **支持撤销和关键帧**
   - 保存原始状态
   - 在 exit 中处理

4. **调试友好**
   - 添加日志
   - 验证数据流

### 10.4. 完整工作流

```
1. 确定数据源
   ├─ Blender 数据结构 → RNA 绑定
   └─ 复杂逻辑 → 自定义函数

2. 实现 getter/setter
   ├─ 读取外部数据
   └─ 写入外部数据

3. 绑定到 gizmo
   ├─ WM_gizmo_target_property_def_*
   └─ 传递 user_data

4. 在 gizmo 回调中使用
   ├─ invoke(): 保存原始值
   ├─ modal(): 读写属性
   ├─ exit(): 确认/撤销
   └─ property_update(): 外部更新

5. 测试和调试
   ├─ 检查绑定
   └─ 追踪数据流
```
