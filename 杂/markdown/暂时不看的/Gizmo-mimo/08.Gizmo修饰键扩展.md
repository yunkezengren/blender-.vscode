## 1. 概述

本章详细介绍如何为 gizmo 系统添加新的修饰键支持，包括：

- **扩展标准系统**：为现有系统添加新修饰键
- **自定义系统**：创建独立的修饰键处理逻辑
- **组合键**：处理多个修饰键同时按下的情况
- **属性驱动**：通过 RNA 属性控制修饰键行为

扩展修饰键时，需要在**键映射层**、**标志传递层**和**gizmo 实现层**三处修改。

## 2. 扩展 Cage2d 的 Ctrl 捕捉

### 2.1. 需求分析

当前 Cage2d 仅支持 Shift 等比缩放，我们希望添加：
- **Ctrl**：网格捕捉
- **Shift+Ctrl**：中心缩放（以中心为枢轴）

### 2.2. 实现步骤

#### 步骤 1: 修改交互数据结构

```cpp
// source/blender/editors/gizmo_library/gizmo_types/cage2d_gizmo.cc

namespace {

struct RectTransformInteraction {
  float orig_mouse[2];
  float orig_matrix_offset[4][4];
  float orig_matrix_final_no_offset[4][4];
  Dial *dial;
  bool use_temp_uniform;      // Shift - 等比缩放
  bool use_snap;              // Ctrl - 网格捕捉
  bool use_center_pivot;      // Shift+Ctrl - 中心枢轴
};

}  // namespace
```

#### 步骤 2: 在 modal 中检测修饰键

```cpp
static wmOperatorStatus gizmo_cage2d_modal(bContext *C,
                                           wmGizmo *gz,
                                           const wmEvent *event,
                                           eWM_GizmoFlagTweak /*tweak_flag*/)
{
  RectTransformInteraction *data = static_cast<RectTransformInteraction *>(gz->interaction_data);
  int transform_flag = RNA_enum_get(gz->ptr, \"transform\");

  // 2.1. 检测 Shift（等比缩放）
  if ((transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM) == 0) {
    const bool use_temp_uniform = (event->modifier & KM_SHIFT) != 0;
    if (use_temp_uniform != data->use_temp_uniform) {
      data->use_temp_uniform = use_temp_uniform;
      // 触发刷新
    }
  }

  // 2.2. 检测 Ctrl（网格捕捉）
  const bool use_snap = (event->modifier & KM_CTRL) != 0;
  if (use_snap != data->use_snap) {
    data->use_snap = use_snap;
    // 触发刷新
  }

  // 2.3. 检测 Shift+Ctrl（中心枢轴）
  const bool use_center = (event->modifier & (KM_SHIFT | KM_CTRL)) == (KM_SHIFT | KM_CTRL);
  if (use_center != data->use_center_pivot) {
    data->use_center_pivot = use_center;
    // 触发刷新
  }

  // 2.4. 投影鼠标位置
  float point_local[2];
  // ... 投影代码

  // 2.5. 应用网格捕捉
  if (data->use_snap) {
    const float grid_size = 10.0f;  // 可配置
    point_local[0] = round(point_local[0] / grid_size) * grid_size;
    point_local[1] = round(point_local[1] / grid_size) * grid_size;
  }

  // 2.6. 根据部件类型处理
  if (gz->highlight_part == ED_GIZMO_CAGE2D_PART_TRANSLATE) {
    // 平移处理
    copy_m4_m4(gz->matrix_offset, data->orig_matrix_offset);

    // 如果使用中心枢轴，不改变位置，只改变尺寸
    if (data->use_center_pivot) {
      // 转换为缩放操作
      gz->highlight_part = ED_GIZMO_CAGE2D_PART_SCALE;
    }
    else {
      // 正常平移
      gz->matrix_offset[3][0] = data->orig_matrix_offset[3][0] +
                                (point_local[0] - data->orig_mouse[0]);
      gz->matrix_offset[3][1] = data->orig_matrix_offset[3][1] +
                                (point_local[1] - data->orig_mouse[1]);
    }
  }
  else if (gz->highlight_part == ED_GIZMO_CAGE2D_PART_SCALE ||
           (gz->highlight_part >= ED_GIZMO_CAGE2D_PART_SCALE_MIN_X &&
            gz->highlight_part <= ED_GIZMO_CAGE2D_PART_SCALE_MAX_X_MAX_Y)) {

    // 缩放处理
    copy_m4_m4(gz->matrix_offset, data->orig_matrix_offset);

    // 2.7. 确定枢轴
    float pivot[2];
    if (data->use_center_pivot) {
      // Shift+Ctrl：强制使用中心枢轴
      zero_v2(pivot);
    }
    else if (transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_TRANSLATE) {
      // 正常：根据部件确定枢轴
      gizmo_pivot_from_scale_part(gz->highlight_part, pivot);
      mul_v2_v2(pivot, dims);
    }
    else {
      // 无平移：中心枢轴
      zero_v2(pivot);
    }

    // 2.8. 计算缩放（包含等比逻辑）
    // ... 缩放计算代码

    if (transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM ||
        data->use_temp_uniform) {
      // 等比缩放
      scale[1] = scale[0] = (scale[1] + scale[0]) / 2.0f;
    }

    // 2.9. 应用缩放
    float matrix_scale[4][4];
    unit_m4(matrix_scale);
    mul_v3_fl(matrix_scale[0], scale[0]);
    mul_v3_fl(matrix_scale[1], scale[1]);
    transform_pivot_set_m4(matrix_scale, float3(UNPACK2(pivot), 0.0f));
    mul_m4_m4_post(gz->matrix_offset, matrix_scale);
  }

  // 2.10. 更新属性
  wmGizmoProperty *gz_prop = WM_gizmo_target_property_find(gz, \"matrix\");
  if (gz_prop->type != nullptr) {
    WM_gizmo_target_property_float_set_array(C, gz, gz_prop, &gz->matrix_offset[0][0]);
  }

  ED_region_tag_redraw_editor_overlays(CTX_wm_region(C));

  return OPERATOR_RUNNING_MODAL;
}
```

### 2.3. 完整代码

```cpp
// 完整的增强版 gizmo_cage2d_modal
static wmOperatorStatus gizmo_cage2d_modal(bContext *C,
                                           wmGizmo *gz,
                                           const wmEvent *event,
                                           eWM_GizmoFlagTweak /*tweak_flag*/)
{
  RectTransformInteraction *data = static_cast<RectTransformInteraction *>(gz->interaction_data);
  int transform_flag = RNA_enum_get(gz->ptr, \"transform\");

  // 1. 检测修饰键状态变化
  bool refresh_needed = false;

  // Shift - 等比缩放
  if ((transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM) == 0) {
    const bool use_temp_uniform = (event->modifier & KM_SHIFT) != 0;
    if (use_temp_uniform != data->use_temp_uniform) {
      data->use_temp_uniform = use_temp_uniform;
      refresh_needed = true;
    }
  }

  // Ctrl - 网格捕捉
  const bool use_snap = (event->modifier & KM_CTRL) != 0;
  if (use_snap != data->use_snap) {
    data->use_snap = use_snap;
    refresh_needed = true;
  }

  // Shift+Ctrl - 中心枢轴
  const bool use_center = (event->modifier & (KM_SHIFT | KM_CTRL)) == (KM_SHIFT | KM_CTRL);
  if (use_center != data->use_center_pivot) {
    data->use_center_pivot = use_center;
    refresh_needed = true;
  }

  // 如果状态改变但不是鼠标移动，返回等待
  if (refresh_needed && event->type != MOUSEMOVE) {
    return OPERATOR_RUNNING_MODAL;
  }

  // 2. 投影鼠标位置
  float point_local[2];
  {
    float matrix_back[4][4];
    copy_m4_m4(matrix_back, gz->matrix_offset);
    copy_m4_m4(gz->matrix_offset, data->orig_matrix_offset);

    bool ok = gizmo_window_project_2d(
        C, gz, blender::float2(blender::int2(event->mval)), 2, false, point_local);

    copy_m4_m4(gz->matrix_offset, matrix_back);

    if (!ok) {
      return OPERATOR_RUNNING_MODAL;
    }
  }

  // 3. 应用网格捕捉
  if (data->use_snap) {
    const float grid_size = 10.0f;  // 可以从属性读取
    point_local[0] = round(point_local[0] / grid_size) * grid_size;
    point_local[1] = round(point_local[1] / grid_size) * grid_size;
  }

  // 4. 获取当前属性值
  wmGizmoProperty *gz_prop = WM_gizmo_target_property_find(gz, \"matrix\");
  if (gz_prop->type != nullptr) {
    WM_gizmo_target_property_float_get_array(gz, gz_prop, &gz->matrix_offset[0][0]);
  }

  // 5. 根据部件处理
  if (gz->highlight_part == ED_GIZMO_CAGE2D_PART_TRANSLATE) {
    // 5.1. 平移处理
    if (data->use_center_pivot) {
      // Shift+Ctrl：转换为缩放
      gz->highlight_part = ED_GIZMO_CAGE2D_PART_SCALE;
      // 继续到缩放处理
    }
    else {
      // 正常平移
      copy_m4_m4(gz->matrix_offset, data->orig_matrix_offset);
      gz->matrix_offset[3][0] = data->orig_matrix_offset[3][0] +
                                (point_local[0] - data->orig_mouse[0]);
      gz->matrix_offset[3][1] = data->orig_matrix_offset[3][1] +
                                (point_local[1] - data->orig_mouse[1]);
      goto apply_property;
    }
  }

  // 5.2. 旋转处理
  if (gz->highlight_part == ED_GIZMO_CAGE2D_PART_ROTATE) {
    // ... 旋转代码（标准实现）
    // 注意：旋转不受修饰键影响
  }
  else {
    // 5.3. 缩放处理
    copy_m4_m4(gz->matrix_offset, data->orig_matrix_offset);

    float dims[2];
    RNA_float_get_array(gz->ptr, \"dimensions\", dims);

    // 确定枢轴
    float pivot[2];
    if (data->use_center_pivot) {
      zero_v2(pivot);  // 中心
    }
    else if (transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_TRANSLATE) {
      gizmo_pivot_from_scale_part(gz->highlight_part, pivot);
      mul_v2_v2(pivot, dims);
    }
    else {
      zero_v2(pivot);  // 中心
    }

    // 旋转鼠标坐标（如果 gizmo 被旋转）
    float curr_mouse[2];
    copy_v2_v2(curr_mouse, data->orig_mouse);

    if (transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_ROTATE) {
      float rot[3][3], loc[3], size[3];
      mat4_to_loc_rot_size(loc, rot, size, gz->matrix_offset);
      invert_m3(rot);

      sub_v2_v2(point_local, loc);
      mul_m3_v2(rot, point_local);
      add_v2_v2(point_local, loc);

      sub_v2_v2(curr_mouse, loc);
      mul_m3_v2(rot, curr_mouse);
      add_v2_v2(curr_mouse, loc);
    }

    // 确定约束轴
    bool constrain_axis[2] = {false};
    gizmo_constrain_from_scale_part(gz->highlight_part, constrain_axis);

    // 计算新尺寸
    float size_new[2], size_orig[2];
    for (int i = 0; i < 2; i++) {
      size_orig[i] = len_v3(data->orig_matrix_offset[i]);
      size_new[i] = size_orig[i];

      if (constrain_axis[i] == false) {
        const float delta_orig = curr_mouse[i] - data->orig_matrix_offset[3][i] -
                                 pivot[i] * size_orig[i];
        const float delta_curr = point_local[i] - data->orig_matrix_offset[3][i] -
                                 pivot[i] * size_orig[i];

        if ((transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_SCALE_SIGNED) == 0) {
          if (signum_i(delta_orig) != signum_i(delta_curr)) {
            size_new[i] = 0.0f;
            continue;
          }
        }

        size_new[i] = delta_curr / (signf(delta_orig) * 0.5f * dims[i] - pivot[i]);
      }
    }

    // 计算缩放比例
    float scale[2] = {1.0f, 1.0f};
    for (int i = 0; i < 2; i++) {
      if (size_orig[i] == 0) {
        size_orig[i] = 1.0f;
        gz->matrix_offset[i][i] = 1.0f;
      }
      scale[i] = size_new[i] / size_orig[i];
    }

    // 处理等比缩放（Shift 或 标志）
    if (transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM ||
        data->use_temp_uniform) {
      if (constrain_axis[0] == false && constrain_axis[1] == false) {
        scale[1] = scale[0] = (scale[1] + scale[0]) / 2.0f;
      }
      else if (constrain_axis[0] == false) {
        scale[1] = scale[0];
      }
      else if (constrain_axis[1] == false) {
        scale[0] = scale[1];
      }
    }

    // 应用缩放
    float matrix_scale[4][4];
    unit_m4(matrix_scale);
    mul_v3_fl(matrix_scale[0], scale[0]);
    mul_v3_fl(matrix_scale[1], scale[1]);
    transform_pivot_set_m4(matrix_scale, float3(UNPACK2(pivot), 0.0f));
    mul_m4_m4_post(gz->matrix_offset, matrix_scale);
  }

apply_property:
  // 6. 更新属性
  if (gz_prop->type != nullptr) {
    WM_gizmo_target_property_float_set_array(C, gz, gz_prop, &gz->matrix_offset[0][0]);
  }

  // 7. 标记重绘
  ED_region_tag_redraw_editor_overlays(CTX_wm_region(C));

  return OPERATOR_RUNNING_MODAL;
}
```

```cpp
// 如果扩展了 tweak_flag 系统

bool shift = (tweak_flag & WM_GIZMO_TWEAK_PRECISE) != 0;
bool ctrl = (tweak_flag & WM_GIZMO_TWEAK_SNAP) != 0;
bool alt = (tweak_flag & WM_GIZMO_TWEAK_ALT) != 0;

if (shift && ctrl) {
  // Shift+Ctrl 组合
}
```

## 6. 完整示例：增强的 Cage2d

### 6.1. 需求规格

| 修饰键 | 功能 | 说明 |
|--------|------|------|
| **无** | 标准变换 | 根据 transform 标志 |
| **Shift** | 等比缩放 | 临时启用 SCALE_UNIFORM |
| **Ctrl** | 网格捕捉 | 捕捉到 10px 网格 |
| **Alt** | 镜像变换 | 水平/垂直镜像 |
| **Shift+Ctrl** | 中心枢轴 | 以中心为枢轴缩放 |
| **Shift+Alt** | 精确镜像 | 捕捉到 45° 镜像 |
| **Ctrl+Alt** | 黄金分割捕捉 | 捕捉到 1.618 比例 |
| **Shift+Ctrl+Alt** | 重置 | 恢复原始状态 |

### 6.2. 实现代码

```cpp
// source/blender/editors/gizmo_library/gizmo_types/cage2d_gizmo_enhanced.cc

namespace {

struct EnhancedRectInteraction {
  float orig_mouse[2];
  float orig_matrix_offset[4][4];
  float orig_matrix_final_no_offset[4][4];
  Dial *dial;

  // 修饰键标志
  bool use_temp_uniform;      // Shift
  bool use_snap;              // Ctrl
  bool use_alt_mode;          // Alt
  bool use_center_pivot;      // Shift+Ctrl
  bool use_golden_ratio;      // Ctrl+Alt
  bool use_reset;             // Shift+Ctrl+Alt
};

}  // namespace

static wmOperatorStatus gizmo_cage2d_enhanced_modal(bContext *C,
                                                    wmGizmo *gz,
                                                    const wmEvent *event,
                                                    eWM_GizmoFlagTweak /*tweak_flag*/)
{
  EnhancedRectInteraction *data = static_cast<EnhancedRectInteraction *>(gz->interaction_data);
  int transform_flag = RNA_enum_get(gz->ptr, \"transform\");

  // 1. 检测所有修饰键组合
  bool shift = (event->modifier & KM_SHIFT) != 0;
  bool ctrl = (event->modifier & KM_CTRL) != 0;
  bool alt = (event->modifier & KM_ALT) != 0;

  // 2. 计算组合标志
  data->use_temp_uniform = shift && !ctrl && !alt;
  data->use_snap = ctrl && !shift && !alt;
  data->use_alt_mode = alt && !shift && !ctrl;
  data->use_center_pivot = shift && ctrl && !alt;
  data->use_golden_ratio = ctrl && alt && !shift;
  data->use_reset = shift && ctrl && alt;

  // 3. 处理重置
  if (data->use_reset) {
    copy_m4_m4(gz->matrix_offset, data->orig_matrix_offset);
    // 更新属性
    wmGizmoProperty *gz_prop = WM_gizmo_target_property_find(gz, \"matrix\");
    if (gz_prop->type != nullptr) {
      WM_gizmo_target_property_float_set_array(C, gz, gz_prop, &data->orig_matrix_offset[0][0]);
    }
    return OPERATOR_RUNNING_MODAL;
  }

  // 4. 投影鼠标
  float point_local[2];
  // ... 投影代码

  // 5. 应用捕捉
  if (data->use_snap) {
    const float grid_size = 10.0f;
    point_local[0] = round(point_local[0] / grid_size) * grid_size;
    point_local[1] = round(point_local[1] / grid_size) * grid_size;
  }
  else if (data->use_golden_ratio) {
    // 黄金分割捕捉 (1.618)
    const float golden = 1.61803398875f;
    point_local[0] = round(point_local[0] / golden) * golden;
    point_local[1] = round(point_local[1] / golden) * golden;
  }

  // 6. 获取属性
  wmGizmoProperty *gz_prop = WM_gizmo_target_property_find(gz, \"matrix\");
  if (gz_prop->type != nullptr) {
    WM_gizmo_target_property_float_get_array(gz, gz_prop, &gz->matrix_offset[0][0]);
  }

  // 7. 根据部件处理
  if (gz->highlight_part == ED_GIZMO_CAGE2D_PART_TRANSLATE) {
    if (data->use_center_pivot) {
      // 转换为缩放
      gz->highlight_part = ED_GIZMO_CAGE2D_PART_SCALE;
    }
    else if (data->use_alt_mode) {
      // Alt+平移 = 镜像
      // 计算镜像方向
      float delta[2] = {
        point_local[0] - data->orig_mouse[0],
        point_local[1] - data->orig_mouse[1]
      };

      // 应用镜像到矩阵
      if (fabs(delta[0]) > fabs(delta[1])) {
        // 水平镜像
        gz->matrix_offset[0][0] = -gz->matrix_offset[0][0];
      }
      else {
        // 垂直镜像
        gz->matrix_offset[1][1] = -gz->matrix_offset[1][1];
      }
      goto apply_property;
    }
    else {
      // 正常平移
      copy_m4_m4(gz->matrix_offset, data->orig_matrix_offset);
      gz->matrix_offset[3][0] = data->orig_matrix_offset[3][0] +
                                (point_local[0] - data->orig_mouse[0]);
      gz->matrix_offset[3][1] = data->orig_matrix_offset[3][1] +
                                (point_local[1] - data->orig_mouse[1]);
      goto apply_property;
    }
  }

  // 8. 缩放处理
  if (gz->highlight_part == ED_GIZMO_CAGE2D_PART_ROTATE) {
    // 旋转（不受修饰键影响，除非 Shift+Alt 精确旋转）
    if (data->use_alt_mode && data->use_temp_uniform) {
      // 精确旋转：限制为 15° 增量
      // ... 旋转代码 ...
    }
    else {
      // 标准旋转
      // ... 旋转代码 ...
    }
  }
  else {
    // 缩放
    copy_m4_m4(gz->matrix_offset, data->orig_matrix_offset);

    float dims[2];
    RNA_float_get_array(gz->ptr, \"dimensions\", dims);

    // 确定枢轴
    float pivot[2];
    if (data->use_center_pivot) {
      zero_v2(pivot);
    }
    else if (transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_TRANSLATE) {
      gizmo_pivot_from_scale_part(gz->highlight_part, pivot);
      mul_v2_v2(pivot, dims);
    }
    else {
      zero_v2(pivot);
    }

    // 旋转坐标
    float curr_mouse[2];
    copy_v2_v2(curr_mouse, data->orig_mouse);

    if (transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_ROTATE) {
      float rot[3][3], loc[3], size[3];
      mat4_to_loc_rot_size(loc, rot, size, gz->matrix_offset);
      invert_m3(rot);

      sub_v2_v2(point_local, loc);
      mul_m3_v2(rot, point_local);
      add_v2_v2(point_local, loc);

      sub_v2_v2(curr_mouse, loc);
      mul_m3_v2(rot, curr_mouse);
      add_v2_v2(curr_mouse, loc);
    }

    // 计算缩放
    bool constrain_axis[2] = {false};
    gizmo_constrain_from_scale_part(gz->highlight_part, constrain_axis);

    float size_new[2], size_orig[2];
    for (int i = 0; i < 2; i++) {
      size_orig[i] = len_v3(data->orig_matrix_offset[i]);
      size_new[i] = size_orig[i];

      if (constrain_axis[i] == false) {
        const float delta_orig = curr_mouse[i] - data->orig_matrix_offset[3][i] -
                                 pivot[i] * size_orig[i];
        const float delta_curr = point_local[i] - data->orig_matrix_offset[3][i] -
                                 pivot[i] * size_orig[i];

        if ((transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_SCALE_SIGNED) == 0) {
          if (signum_i(delta_orig) != signum_i(delta_curr)) {
            size_new[i] = 0.0f;
            continue;
          }
        }

        size_new[i] = delta_curr / (signf(delta_orig) * 0.5f * dims[i] - pivot[i]);
      }
    }

    // 计算比例
    float scale[2] = {1.0f, 1.0f};
    for (int i = 0; i < 2; i++) {
      if (size_orig[i] == 0) {
        size_orig[i] = 1.0f;
        gz->matrix_offset[i][i] = 1.0f;
      }
      scale[i] = size_new[i] / size_orig[i];
    }

    // 应用等比（Shift 或 标志）
    bool uniform = (transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM) ||
                   data->use_temp_uniform;

    if (uniform) {
      if (constrain_axis[0] == false && constrain_axis[1] == false) {
        scale[1] = scale[0] = (scale[1] + scale[0]) / 2.0f;
      }
      else if (constrain_axis[0] == false) {
        scale[1] = scale[0];
      }
      else if (constrain_axis[1] == false) {
        scale[0] = scale[1];
      }
    }

    // Alt 模式：镜像缩放
    if (data->use_alt_mode) {
      if (constrain_axis[0] == false) {
        scale[0] = -scale[0];  // 水平镜像
      }
      if (constrain_axis[1] == false) {
        scale[1] = -scale[1];  // 垂直镜像
      }
    }

    // 应用缩放
    float matrix_scale[4][4];
    unit_m4(matrix_scale);
    mul_v3_fl(matrix_scale[0], scale[0]);
    mul_v3_fl(matrix_scale[1], scale[1]);
    transform_pivot_set_m4(matrix_scale, float3(UNPACK2(pivot), 0.0f));
    mul_m4_m4_post(gz->matrix_offset, matrix_scale);
  }

apply_property:
  // 9. 更新属性
  if (gz_prop->type != nullptr) {
    WM_gizmo_target_property_float_set_array(C, gz, gz_prop, &gz->matrix_offset[0][0]);
  }

  // 10. 标记重绘
  ED_region_tag_redraw_editor_overlays(CTX_wm_region(C));

  return OPERATOR_RUNNING_MODAL;
}
```

## 7. 最佳实践

### 7.1. 设计原则

#### ✅ 推荐

1. **保持一致性**：遵循 Shift=精确/等比，Ctrl=捕捉 的惯例
2. **渐进增强**：基础功能不依赖修饰键
3. **清晰反馈**：通过颜色、提示或辅助线显示修饰键状态
4. **可配置**：允许用户通过偏好设置调整修饰键行为
5. **文档化**：在代码注释中说明修饰键功能

#### ❌ 避免

1. **过度使用**：不要为每个功能都分配修饰键
2. **冲突**：避免与系统快捷键冲突
3. **不可发现**：用户应该能猜到修饰键功能
4. **硬编码**：尽量使用配置而非硬编码键值
5. **不清理**：确保交互数据正确释放

### 7.2. 性能考虑

```cpp
// 优化：避免不必要的计算
```

## 8. 总结

### 8.1. 扩展检查表

在扩展修饰键时，确保修改以下位置：

- [ ] **枚举定义**：`eWM_GizmoFlagTweak`（如果使用标准系统）
- [ ] **模态数据**：交互数据结构
- [ ] **键映射**：`wm_gizmogroup_tweak_modal_keymap`
- [ ] **模态处理**：`gizmo_tweak_modal` 标志设置
- [ ] **Gizmo 回调**：`gizmo_modal` 参数和处理
- [ ] **辅助函数**：`gizmo_transform_flag_get` 等
- [ ] **绘制**：视觉反馈（可选）
- [ ] **测试**：所有组合键场景

### 8.2. 两种扩展方式对比

| 方式 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| **扩展标准系统** | 统一、可配置、所有 gizmo 受益 | 需要修改核心代码 | 全局功能（如 Alt） |
| **自定义系统** | 灵活、独立、快速迭代 | 仅限特定 gizmo | 特殊需求（如 Cage2d 等比） |

### 8.3. 推荐方案

对于用户需求：

1. **Cage2d Ctrl 捕捉**：使用自定义系统（已有 Shift 等比）
2. **Cage2d Alt 镜像**：使用自定义系统
3. **全局 Alt 功能**：扩展标准系统（所有 gizmo 受益）

### 8.4. 完整实现流程

```
1. 需求分析 → 确定功能和组合键
2. 选择方案 → 标准系统 vs 自定义系统
3. 修改数据结构 → 添加标志字段
4. 实现检测逻辑 → modal 中处理事件
5. 实现功能逻辑 → 变换计算
6. 添加反馈 → 绘制辅助信息
7. 测试验证 → 所有场景
8. 文档更新 → 注释和说明
```
