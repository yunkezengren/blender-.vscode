## 概述

正确的标志组合对于 gizmo 的行为至关重要。本章将通过实际代码示例，展示不同场景下的最佳标志组合。

<span style="background-color: #2E7D32; color: white; padding: 2px 6px; border-radius: 3px;">★ Insight</span>
标志组合的核心原则：**功能需求** → **变换能力** → **绘制选项** → **组行为**

## 2. 节点编辑器标志组合

### 2.1. 背景变换 (Backdrop Transform)

**源码**: `source/blender/editors/space_node/node_gizmo.cc:150-216`

**需求**:
- 仅平移和等比缩放背景
- 持久化显示
- 不需要旋转

**标志组合**:
```cpp
/* Gizmo 类型: GIZMO_GT_cage_2d */

/* 1. 变换能力 */
RNA_enum_set(gz->ptr, "transform",
  ED_GIZMO_CAGE_XFORM_FLAG_TRANSLATE |      // 平移
  ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM);  // 等比缩放

/* 2. 绘制选项 */
RNA_enum_set(gz->ptr, "draw_options",
  ED_GIZMO_CAGE_DRAW_FLAG_XFORM_CENTER_HANDLE); // 中心手柄

/* 3. 个体标志 */
gz->flag = WM_GIZMO_DRAW_MODAL;  // 拖拽时显示

/* 4. 组标志 */
gzgroup->flag = WM_GIZMOGROUPTYPE_PERSISTENT;  // 持久化
```

**完整代码**:
```cpp
static void WIDGETGROUP_node_transform_setup(const bContext * /*C*/, wmGizmoGroup *gzgroup)
{
  wmGizmoWrapper *wwrapper = MEM_mallocN<wmGizmoWrapper>(__func__);
  wwrapper->gizmo = WM_gizmo_new("GIZMO_GT_cage_2d", gzgroup, nullptr);

  RNA_enum_set(wwrapper->gizmo->ptr, "transform",
               ED_GIZMO_CAGE_XFORM_FLAG_TRANSLATE |
               ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM);

  gzgroup->customdata = wwrapper;
}

void NODE_GGT_backdrop_transform(wmGizmoGroupType *gzgt)
{
  gzgt->name = "Backdrop Transform Widget";
  gzgt->idname = "NODE_GGT_backdrop_transform";
  gzgt->flag |= WM_GIZMOGROUPTYPE_PERSISTENT;
  gzgt->poll = WIDGETGROUP_node_transform_poll;
  gzgt->setup = WIDGETGROUP_node_transform_setup;
  gzgt->setup_keymap = WM_gizmogroup_setup_keymap_generic_maybe_drag;
  gzgt->refresh = WIDGETGROUP_node_transform_refresh;
}
```

**定义位置**: `source/blender/editors/space_node/node_gizmo.cc:150-216`

## 5. 通用标志组合模式

### 5.1. 持久化 gizmo

**适用场景**: 节点编辑器、图像编辑器等

```cpp
/* 组标志 */
gzgroup->flag = WM_GIZMOGROUPTYPE_PERSISTENT;

/* 个体标志 */
gz->flag = WM_GIZMO_DRAW_MODAL;

/* 键映射 */
gzgt->setup_keymap = WM_gizmogroup_setup_keymap_generic_maybe_drag;
```

**特点**:
- ✅ 在文件加载之间保持
- ✅ 拖拽时显示
- ✅ 支持点击或拖拽

---

### 5.2. 临时 gizmo

**适用场景**: 工具激活时显示的 gizmo

```cpp
/* 组标志 */
gzgroup->flag = WM_GIZMOGROUPTYPE_TOOL_INIT;

/* 个体标志 */
gz->flag = WM_GIZMO_DRAW_HOVER | WM_GIZMO_DRAW_MODAL;

/* 键映射 */
gzgt->setup_keymap = WM_gizmogroup_setup_keymap_generic_drag;
```

**特点**:
- ✅ 仅在工具激活时显示
- ✅ 悬停和拖拽时显示
- ✅ 拖拽专用键映射

---

### 5.3. 3D 空间 gizmo

**适用场景**: 3D 视图中的 gizmo

```cpp
/* 组标志 */
gzgroup->flag = WM_GIZMOGROUPTYPE_3D |
                WM_GIZMOGROUPTYPE_SCALE |
                WM_GIZMOGROUPTYPE_SELECT |
                WM_GIZMOGROUPTYPE_PERSISTENT;

/* 个体标志 */
gz->flag = WM_GIZMO_DRAW_MODAL | WM_GIZMO_NEEDS_UNDO;

/* 键映射 */
gzgt->setup_keymap = WM_gizmogroup_setup_keymap_generic_maybe_drag;
```

**特点**:
- ✅ 3D 空间渲染
- ✅ 随视图缩放
- ✅ 可选择
- ✅ 需要撤销

---

### 5.4. 精确交互

**适用场景**: 需要避免光标下闪烁的 gizmo

```cpp
/* 组标志 */
gzgroup->flag = WM_GIZMOGROUPTYPE_PERSISTENT |
                WM_GIZMOGROUPTYPE_DELAY_REFRESH_FOR_TWEAK;

/* 个体标志 */
gz->flag = WM_GIZMO_DRAW_MODAL |
           WM_GIZMO_NEEDS_UNDO |
           WM_GIZMO_EVENT_HANDLE_ALL;

/* 在 refresh 回调中 */
static void my_gizmo_refresh(const bContext *C, wmGizmoGroup *gzgroup)
{
  if (WM_gizmomap_tag_delay_refresh_for_tweak_check(gzgroup->parent_gzmap)) {
    return;  // 延迟刷新
  }
  // 正常刷新逻辑
}
```

**特点**:
- ✅ 避免拖拽时闪烁
- ✅ 捕获所有事件
- ✅ 需要撤销

## 6. 标志组合决策树

```
开始
│
├─ 是否在 3D 空间？
│  ├─ 是 → 添加 WM_GIZMOGROUPTYPE_3D
│  ├─ 否 → 2D 空间
│
├─ 是否需要随视图缩放？
│  ├─ 是 → 添加 WM_GIZMOGROUPTYPE_SCALE
│  └─ 否 → 保持固定大小
│
├─ 是否需要深度剔除？
│  ├─ 是 → 添加 WM_GIZMOGROUPTYPE_DEPTH_3D
│  └─ 否 → 始终显示
│
├─ 是否需要选择？
│  ├─ 是 → 添加 WM_GIZMOGROUPTYPE_SELECT
│  └─ 否 → 不可选择
│
├─ 是否持久化？
│  ├─ 是 → 添加 WM_GIZMOGROUPTYPE_PERSISTENT
│  └─ 否 → 临时显示
│
├─ 是否延迟刷新？
│  ├─ 是 → 添加 WM_GIZMOGROUPTYPE_DELAY_REFRESH_FOR_TWEAK
│  └─ 否 → 立即刷新
│
├─ 个体标志
│  ├─ 拖拽时显示 → WM_GIZMO_DRAW_MODAL
│  ├─ 悬停时显示 → WM_GIZMO_DRAW_HOVER
│  ├─ 需要撤销 → WM_GIZMO_NEEDS_UNDO
│  └─ 锁定光标 → WM_GIZMO_MOVE_CURSOR
│
└─ 变换能力 (Cage)
   ├─ 平移 → ED_GIZMO_CAGE_XFORM_FLAG_TRANSLATE
   ├─ 旋转 → ED_GIZMO_CAGE_XFORM_FLAG_ROTATE
   ├─ 缩放 → ED_GIZMO_CAGE_XFORM_FLAG_SCALE
   ├─ 等比 → ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM
   └─ 负缩放 → ED_GIZMO_CAGE_XFORM_FLAG_SCALE_SIGNED
```

## 7. 常见错误

### ❌ 错误 1: 忘记持久化标志

```cpp
// 错误：gizmo 在文件加载后消失
gzgroup->flag = 0;  // 缺少 WM_GIZMOGROUPTYPE_PERSISTENT

// 正确
gzgroup->flag = WM_GIZMOGROUPTYPE_PERSISTENT;
```

---

### ❌ 错误 2: 3D gizmo 缺少 3D 标志

```cpp
// 错误：在 3D 视图中显示不正确
gzgroup->flag = WM_GIZMOGROUPTYPE_PERSISTENT;

// 正确
gzgroup->flag = WM_GIZMOGROUPTYPE_3D |
                WM_GIZMOGROUPTYPE_SCALE |
                WM_GIZMOGROUPTYPE_PERSISTENT;
```

---

### ❌ 错误 3: 变换标志不匹配

```cpp
// 错误：需要旋转但未启用
RNA_enum_set(gz->ptr, "transform",
  ED_GIZMO_CAGE_XFORM_FLAG_TRANSLATE |
  ED_GIZMO_CAGE_XFORM_FLAG_SCALE);  // 缺少 ROTATE

// 正确
RNA_enum_set(gz->ptr, "transform",
  ED_GIZMO_CAGE_XFORM_FLAG_TRANSLATE |
  ED_GIZMO_CAGE_XFORM_FLAG_ROTATE |
  ED_GIZMO_CAGE_XFORM_FLAG_SCALE);
```

---

### ❌ 错误 4: 忘记设置键映射

```cpp
// 错误：gizmo 不响应鼠标
gzgt->setup_keymap = nullptr;  // 或忘记设置

// 正确
gzgt->setup_keymap = WM_gizmogroup_setup_keymap_generic_maybe_drag;
```

---

### ❌ 错误 5: 个体标志与组标志混淆

```cpp
// 错误：在组上设置个体标志
gzgroup->flag = WM_GIZMO_DRAW_MODAL;  // 错误！

// 正确
gzgroup->flag = WM_GIZMOGROUPTYPE_PERSISTENT;
gz->flag = WM_GIZMO_DRAW_MODAL;  // 在 gizmo 上设置
```

## 8. 总结

### 标志组合速查表

| 场景 | 组标志 | 个体标志 | 变换标志 | 绘制选项 |
|------|--------|----------|----------|----------|
| **背景变换** | `PERSISTENT` | `DRAW_MODAL` | `TRANSLATE \| SCALE_UNIFORM` | `XFORM_CENTER_HANDLE` |
| **裁剪** | `PERSISTENT` | `DRAW_MODAL` | `TRANSLATE \| SCALE` | 默认 |
| **遮罩** | `PERSISTENT` | `DRAW_MODAL` | `TRANSLATE \| ROTATE \| SCALE` | `CENTER_HANDLE \| CORNER_HANDLES` |
| **分割** | `PERSISTENT` | `DRAW_MODAL` | `TRANSLATE \| ROTATE` | `NOP` |
| **3D 变换** | `3D \| SCALE \| PERSISTENT` | `DRAW_MODAL \| NEEDS_UNDO` | `TRANSLATE \| ROTATE \| SCALE` | 默认 |
| **临时工具** | `TOOL_INIT` | `DRAW_HOVER \| DRAW_MODAL` | 根据需求 | 根据需求 |
| **精确交互** | `PERSISTENT \| DELAY_REFRESH` | `DRAW_MODAL \| NEEDS_UNDO \| EVENT_HANDLE_ALL` | 根据需求 | 根据需求 |

---

### 关键原则

1. **功能优先**: 先确定需要什么功能（平移、旋转、缩放）
2. **空间匹配**: 3D 空间必须使用 `WM_GIZMOGROUPTYPE_3D`
3. **持久化**: 编辑器 gizmo 使用 `WM_GIZMOGROUPTYPE_PERSISTENT`
4. **撤销支持**: 用户修改数据的 gizmo 使用 `WM_GIZMO_NEEDS_UNDO`
5. **延迟刷新**: 避免闪烁时使用 `WM_GIZMOGROUPTYPE_DELAY_REFRESH_FOR_TWEAK`
