## 概述

Blender 的 gizmo 系统使用多种标志 (flags) 来控制行为、外观和交互。这些标志分为几个不同的类别，每个类别影响 gizmo 的不同方面。

<span style="background-color: #2E7D32; color: white; padding: 2px 6px; border-radius: 3px;">★ Insight</span>
标志使用位运算 (bitwise operations) 进行组合。例如：`flag = WM_GIZMO_DRAW_HOVER | WM_GIZMO_DRAW_MODAL`

**源码位置**: `source/blender/windowmanager/gizmo/WM_gizmo_types.hh`

---

## 2. 个体 Gizmo 标志 (eWM_GizmoFlag)

这些标志直接影响单个 gizmo 的行为。

### 2.1. 绘制控制标志

| 标志 | 值 | 说明 | 使用场景 |
|------|----|------|----------|
| **WM_GIZMO_DRAW_HOVER** | `(1 << 0)` | 仅在悬停时绘制 | 隐藏状态，只在鼠标悬停时显示 |
| **WM_GIZMO_DRAW_MODAL** | `(1 << 1)` | 拖拽时绘制 | 交互过程中保持显示 |
| **WM_GIZMO_DRAW_VALUE** | `(1 << 2)` | 拖拽时显示当前值 | 显示数值指示器 |

**源码位置**: `source/blender/windowmanager/gizmo/WM_gizmo_types.hh:54-92`

**示例**:
```cpp
// 仅在悬停时显示的 gizmo
gz->flag |= WM_GIZMO_DRAW_HOVER;

// 拖拽时显示的 gizmo（常用于变换操作）
gz->flag |= WM_GIZMO_DRAW_MODAL;
```

---

### 2.2. 可见性与选择标志

| 标志 | 值 | 说明 | 使用场景 |
|------|----|------|----------|
| **WM_GIZMO_HIDDEN** | `(1 << 3)` | 完全隐藏 | 临时禁用显示 |
| **WM_GIZMO_HIDDEN_SELECT** | `(1 << 4)` | 隐藏但可选择 | 不可见但可交互 |
| **WM_GIZMO_HIDDEN_KEYMAP** | `(1 << 5)` | 忽略键映射 | 禁用键盘快捷键 |

**示例**:
```cpp
// 隐藏 gizmo 但保持可选择
gz->flag |= WM_GIZMO_HIDDEN_SELECT;

// 临时隐藏
WM_gizmo_set_flag(gz, WM_GIZMO_HIDDEN, true);
```

---

### 2.3. 绘制缩放标志

| 标志 | 值 | 说明 | 使用场景 |
|------|----|------|----------|
| **WM_GIZMO_DRAW_OFFSET_SCALE** | `(1 << 6)` | 偏移随缩放变化 | 屏幕空间定位 |
| **WM_GIZMO_DRAW_NO_SCALE** | `(1 << 7)` | 跳过缩放计算 | 界面元素对齐 |

**源码位置**: `source/blender/windowmanager/gizmo/WM_gizmo_types.hh:65-73`

**详细说明**:
- `WM_GIZMO_DRAW_OFFSET_SCALE`: 当 offset 用于避免屏幕空间重叠而不是绝对定位时使用
- `WM_GIZMO_DRAW_NO_SCALE`: 用户仍应使用 `scale_final` 处理手柄，但计算最终矩阵时跳过缩放

**示例**:
```cpp
// 界面下方的 gizmo，需要与界面元素对齐
gz->flag |= WM_GIZMO_DRAW_NO_SCALE;
```

---

### 2.4. 交互控制标志

| 标志 | 值 | 说明 | 使用场景 |
|------|----|------|----------|
| **WM_GIZMO_MOVE_CURSOR** | `(1 << 8)` | 隐藏并锁定光标 | 精确拖拽操作 |
| **WM_GIZMO_SELECT_BACKGROUND** | `(1 << 9)` | 不写入深度缓冲 | 背景选择 |
| **WM_GIZMO_OPERATOR_TOOL_INIT** | `(1 << 10)` | 使用工具的算子属性 | 工具集成 |
| **WM_GIZMO_EVENT_HANDLE_ALL** | `(1 << 11)` | 不传递事件 | 防止事件被其他处理器窃取 |
| **WM_GIZMO_NO_TOOLTIP** | `(1 << 12)` | 禁用工具提示 | 避免干扰 |
| **WM_GIZMO_NEEDS_UNDO** | `(1 << 13)` | 每次使用后推入撤销 | 需要撤销支持 |

**源码位置**: `source/blender/windowmanager/gizmo/WM_gizmo_types.hh:74-92`

**示例**:
```cpp
// 需要撤销支持的变换 gizmo
gz->flag |= WM_GIZMO_NEEDS_UNDO;

// 精确操作（锁定光标）
gz->flag |= WM_GIZMO_MOVE_CURSOR;

// 防止事件传递
gz->flag |= WM_GIZMO_EVENT_HANDLE_ALL;
```

## 3. Gizmo 组类型标志 (eWM_GizmoFlagGroupTypeFlag)

这些标志影响整个 gizmo 组的行为。

### 3.1. 空间类型标志

| 标志 | 值 | 说明 | 使用场景 |
|------|----|------|----------|
| **WM_GIZMOGROUPTYPE_3D** | `(1 << 0)` | 标记为 3D gizmo 组 | 3D 视图中的 gizmo |
| **WM_GIZMOGROUPTYPE_SCALE** | `(1 << 1)` | 随视图缩放 | 3D 对象缩放 |
| **WM_GIZMOGROUPTYPE_DEPTH_3D** | `(1 << 2)` | 深度剔除 | 被其他几何体遮挡时隐藏 |

**源码位置**: `source/blender/windowmanager/gizmo/WM_gizmo_types.hh:99-157`

**示例**:
```cpp
void MY_GT_my_group(wmGizmoGroupType *gzgt)
{
  gzgt->flag = WM_GIZMOGROUPTYPE_3D | WM_GIZMOGROUPTYPE_SCALE;
}
```

---

### 3.2. 选择与持久性标志

| 标志 | 值 | 说明 | 使用场景 |
|------|----|------|----------|
| **WM_GIZMOGROUPTYPE_SELECT** | `(1 << 3)` | 允许选择 | 用户可点击选择 |
| **WM_GIZMOGROUPTYPE_PERSISTENT** | `(1 << 4)` | 保持在文件加载之间 | 持久化 gizmo |

**示例**:
```cpp
// 节点编辑器的 gizmo 组（持久化）
gzgt->flag |= WM_GIZMOGROUPTYPE_PERSISTENT;
```

---

### 3.3. 模态绘制标志

| 标志 | 值 | 说明 | 使用场景 |
|------|----|------|----------|
| **WM_GIZMOGROUPTYPE_DRAW_MODAL_ALL** | `(1 << 5)` | 交互时显示所有 gizmo | 多 gizmo 协作 |
| **WM_GIZMOGROUPTYPE_DRAW_MODAL_EXCLUDE** | `(1 << 6)` | 模态时隐藏此组 | 避免干扰 |

---

### 3.4. 工具集成标志

| 标志 | 值 | 说明 | 使用场景 |
|------|----|------|----------|
| **WM_GIZMOGROUPTYPE_TOOL_INIT** | `(1 << 7)` | 仅在工具激活时 | 工具专用 gizmo |
| **WM_GIZMOGROUPTYPE_TOOL_FALLBACK_KEYMAP** | `(1 << 8)` | 支持回退键映射 | 工具键映射兼容 |
| **WM_GIZMOGROUPTYPE_DELAY_REFRESH_FOR_TWEAK** | `(1 << 9)` | 延迟刷新 | 避免光标下闪烁 |
| **WM_GIZMOGROUPTYPE_VR_REDRAWS** | `(1 << 10)` | 持续重绘 | VR 环境 |

**源码位置**: `source/blender/windowmanager/gizmo/WM_gizmo_types.hh:140-156`

**延迟刷新详解**:
```cpp
// 在 refresh 回调中使用
if (WM_gizmomap_tag_delay_refresh_for_tweak_check(gzgroup->parent_gzmap)) {
  // 延迟刷新，避免在拖拽时 gizmo 在光标下闪烁
  return;
}
```

## 4. 变换标志 (Cage Gizmo)

**源码**: `source/blender/editors/include/ED_gizmo_library.hh:93-99`

### 4.1. 变换能力标志

| 标志 | 值 | 说明 | 使用场景 |
|------|----|------|----------|
| **ED_GIZMO_CAGE_XFORM_FLAG_TRANSLATE** | `(1 << 0)` | 平移 | 移动操作 |
| **ED_GIZMO_CAGE_XFORM_FLAG_ROTATE** | `(1 << 1)` | 旋转 | 旋转操作 |
| **ED_GIZMO_CAGE_XFORM_FLAG_SCALE** | `(1 << 2)` | 缩放 | 缩放操作 |
| **ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM** | `(1 << 3)` | 等比缩放 | 保持比例 |
| **ED_GIZMO_CAGE_XFORM_FLAG_SCALE_SIGNED** | `(1 << 4)` | 允许负缩放 | 反向缩放 |

**组合示例**:
```cpp
// 仅平移和等比缩放（背景变换）
RNA_enum_set(gz->ptr, "transform",
  ED_GIZMO_CAGE_XFORM_FLAG_TRANSLATE | ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM);

// 全功能（遮罩）
RNA_enum_set(gz->ptr, "transform",
  ED_GIZMO_CAGE_XFORM_FLAG_TRANSLATE |
  ED_GIZMO_CAGE_XFORM_FLAG_ROTATE |
  ED_GIZMO_CAGE_XFORM_FLAG_SCALE);

// 仅旋转（分割）
RNA_enum_set(gz->ptr, "transform",
  ED_GIZMO_CAGE_XFORM_FLAG_TRANSLATE |
  ED_GIZMO_CAGE_XFORM_FLAG_ROTATE);
```

## 7. 调整标志 (eWM_GizmoFlagTweak)

**源码**: `source/blender/windowmanager/gizmo/WM_gizmo_types.hh:195-200`

这些标志通过修饰键传递给 gizmo 的 `modal` 回调。

| 标志 | 值 | 修饰键 | 说明 |
|------|----|--------|------|
| **WM_GIZMO_TWEAK_PRECISE** | `(1 << 0)` | Shift | 精确模式（降低灵敏度） |
| **WM_GIZMO_TWEAK_SNAP** | `(1 << 1)` | Ctrl | 捕捉模式 |

**源码位置**: `source/blender/windowmanager/gizmo/intern/wm_gizmo_group.cc:504-517`

**处理示例**:
```cpp
static wmOperatorStatus gizmo_modal(bContext *C, wmGizmo *gz,
                                    const wmEvent *event,
                                    eWM_GizmoFlagTweak tweak_flag)
{
  // 检查精确模式
  bool use_precision = (tweak_flag & WM_GIZMO_TWEAK_PRECISE) != 0;
  float sensitivity = use_precision ? 0.01f : 0.1f;

  // 检查捕捉模式
  bool use_snap = (tweak_flag & WM_GIZMO_TWEAK_SNAP) != 0;
  if (use_snap) {
    // 应用捕捉逻辑
  }

  // 处理交互...
  return OPERATOR_RUNNING_MODAL;
}
```

**详细说明**: 详见 `07_Gizmo_Modifier_Keys.md`

## 8. 标志的使用场景

### 8.1. 节点编辑器背景变换

```cpp
// source/blender/editors/space_node/node_gizmo.cc:150-161
static void WIDGETGROUP_node_transform_setup(const bContext * /*C*/, wmGizmoGroup *gzgroup)
{
  wmGizmoWrapper *wwrapper = MEM_mallocN<wmGizmoWrapper>(__func__);
  wwrapper->gizmo = WM_gizmo_new("GIZMO_GT_cage_2d", gzgroup, nullptr);

  // 只允许平移和等比缩放
  RNA_enum_set(wwrapper->gizmo->ptr, "transform",
               ED_GIZMO_CAGE_XFORM_FLAG_TRANSLATE |
               ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM);

  // 组持久化
  gzgroup->flag |= WM_GIZMOGROUPTYPE_PERSISTENT;

  gzgroup->customdata = wwrapper;
}
```

## 9. 总结

### 标志分类速查

| 类别 | 标志前缀 | 作用范围 | 常用组合 |
|------|----------|----------|----------|
| **个体行为** | `WM_GIZMO_` | 单个 gizmo | `DRAW_MODAL \| NEEDS_UNDO` |
| **组行为** | `WM_GIZMOGROUPTYPE_` | 整个 gizmo 组 | `PERSISTENT \| 3D` |
| **变换能力** | `ED_GIZMO_CAGE_XFORM_` | Cage gizmo | `TRANSLATE \| SCALE` |
| **绘制选项** | `ED_GIZMO_*_DRAW_FLAG_` | 特定 gizmo 类型 | `CORNER_HANDLES` |
| **状态** | `WM_GIZMO_STATE_` | 运行时状态 | 只读 |
| **调整** | `WM_GIZMO_TWEAK_` | 交互时修饰键 | 通过 modal 回调接收 |

### 关键标志组合

**简单变换**:
```cpp
gz->flag = WM_GIZMO_DRAW_MODAL | WM_GIZMO_NEEDS_UNDO;
gzgroup->flag = WM_GIZMOGROUPTYPE_PERSISTENT;
```

**3D 视图**:
```cpp
gzgroup->flag = WM_GIZMOGROUPTYPE_3D | WM_GIZMOGROUPTYPE_SCALE;
```

**延迟刷新**:
```cpp
gzgroup->flag = WM_GIZMOGROUPTYPE_DELAY_REFRESH_FOR_TWEAK;
```
