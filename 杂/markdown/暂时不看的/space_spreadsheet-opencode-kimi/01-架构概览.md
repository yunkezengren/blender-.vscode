## 2. 核心设计模式

### 2.1 策略模式 - DataSource

```mermaid
classDiagram
    class DataSource {
        <<abstract>>
        +foreach_default_column_ids(fn)
        +get_column_values(column_id)*
        +tot_rows()*
        +has_selection_filter()*
    }

    class GeometryDataSource {
        -Object* object_orig_
        -GeometrySet geometry_set_
        -GeometryComponent* component_
        -AttrDomain domain_
        +get_column_values(column_id) override
        +tot_rows() override
        +has_selection_filter() override
    }

    class VolumeDataSource {
        -GeometrySet geometry_set_
        -VolumeComponent* component_
        +get_column_values(column_id) override
        +tot_rows() override
    }

    class ListDataSource {
        -ListPtr list_
        +get_column_values(column_id) override
        +tot_rows() override
    }

    class BundleDataSource {
        -BundlePtr bundle_
        -Vector~string~ flat_item_keys_
        +get_column_values(column_id) override
        +tot_rows() override
    }

    DataSource <|-- GeometryDataSource
    DataSource <|-- VolumeDataSource
    DataSource <|-- VolumeGridDataSource
    DataSource <|-- ListDataSource
    DataSource <|-- BundleDataSource
    DataSource <|-- ClosureSignatureDataSource
    DataSource <|-- SingleValueDataSource
```

### 2.2 访问者模式 - 列值遍历

```mermaid
sequenceDiagram
    participant Client as 调用方
    participant GDS as GeometryDataSource
    participant Col as SpreadsheetColumnID
    participant CV as ColumnValues

    Client->>GDS: foreach_default_column_ids(fn)
    loop 遍历所有属性
        GDS->>Col: 创建列ID
        GDS->>Client: fn(column_id, is_extra)
    end
    Client->>GDS: get_column_values(column_id)
    GDS->>CV: 创建ColumnValues
    GDS-->>Client: return unique_ptr~ColumnValues~
```

## 6. 关键常量和配置

| 常量 | 值 | 含义 |
|-----|-----|-----|
| `SPREADSHEET_EDGE_ACTION_ZONE` | `UI_UNIT_X * 0.3f` | 列边缘拖拽触发区域 |
| `SPREADSHEET_WIDTH_UNIT` | 隐藏常量 | 列宽单位 |
| `SPREADSHEET_COLUMN_FLAG_UNAVAILABLE` | 标志位 | 列不可用标记 |
| `SPREADSHEET_FILTER_ENABLE` | 标志位 | 启用筛选 |

## 7. 学习建议

### 阅读顺序
1. ⭐ `spreadsheet_data_source.hh` - 理解抽象接口
2. ⭐ `spreadsheet_data_source_geometry.hh` - 理解具体实现
3. `spreadsheet_column_values.hh` - 值系统
4. `spreadsheet_draw.hh` - 绘制系统
5. `space_spreadsheet.cc` - 整合逻辑

### 重点关注
- DataSource 如何解耦数据获取和UI显示
- ColumnValues 如何统一不同类型数据的表示
- SpreadsheetDrawer 如何高效渲染大量数据
