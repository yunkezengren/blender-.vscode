# Blender 电子表格行过滤系统

## 概述

行过滤系统允许用户根据条件筛选显示的行，支持多种数据类型和比较操作。

## 过滤器结构

### SpreadsheetRowFilter

```cpp
/* 来源: DNA_space_types.h */
typedef struct SpreadsheetRowFilter {
  struct SpreadsheetRowFilter *next, *prev;

  /* 过滤器配置 */
  char column_name[64];  /* 要过滤的列名 */
  short operation;       /* eSpreadsheetFilterOperation */
  short flag;            /* 标志位 */

  /* 阈值 (用于浮点数的距离比较) */
  float threshold;

  /* 值存储 (根据数据类型使用不同字段) */
  int value_int;
  float value_float;
  int2 value_int2;
  int3 value_int3;
  float2 value_float2;
  float3 value_float3;
  float4 value_color;  /* ColorGeometry4f 的浮点表示 */
  char *value_string;  /* 动态分配的字符串 */

  /* 用于 UI 展开/折叠状态 */
  short ui_flag;
} SpreadsheetRowFilter;
```

```cpp
SpreadsheetRowFilter *spreadsheet_row_filter_new()
{
  SpreadsheetRowFilter *row_filter = MEM_callocN<SpreadsheetRowFilter>(__func__);

  /* 默认设置 */
  row_filter->flag = (SPREADSHEET_ROW_FILTER_UI_EXPAND | SPREADSHEET_ROW_FILTER_ENABLED);
  row_filter->operation = SPREADSHEET_ROW_FILTER_LESS;
  row_filter->threshold = 0.01f;
  row_filter->column_name[0] = '\0';

  return row_filter;
}
```

### 复制过滤器

```cpp
/* 来源: spreadsheet_row_filter.cc */
SpreadsheetRowFilter *spreadsheet_row_filter_copy(const SpreadsheetRowFilter *src_row_filter)
{
  SpreadsheetRowFilter *new_filter = spreadsheet_row_filter_new();

  /* 浅拷贝所有字段 */
  memcpy(new_filter, src_row_filter, sizeof(SpreadsheetRowFilter));

  /* 重置链表指针 */
  new_filter->next = nullptr;
  new_filter->prev = nullptr;

  return new_filter;
}
```

## 过滤器应用流程

### 主过滤函数

```cpp
/* 来源: spreadsheet_row_filter.cc */
IndexMask spreadsheet_filter_rows(const SpaceSpreadsheet &sspreadsheet,
                                  const SpreadsheetLayout &spreadsheet_layout,
                                  const DataSource &data_source,
                                  ResourceScope &scope)
{
  const int tot_rows = data_source.tot_rows();

  /* 1. 检查是否需要过滤 */
  const bool use_selection = use_selection_filter(sspreadsheet, data_source);
  const bool use_filters = use_row_filters(sspreadsheet);

  /* 2. 如果不需要过滤，返回所有行 */
  if (!(use_selection || use_filters)) {
    return IndexMask(tot_rows);
  }

  /* 3. 获取内存池 */
  IndexMaskMemory &mask_memory = scope.construct<IndexMaskMemory>();
  IndexMask mask(tot_rows);

  /* 4. 应用选择过滤 */
  if (use_selection) {
    const GeometryDataSource *geometry_data_source = dynamic_cast<const GeometryDataSource *>(
        &data_source);
    mask = geometry_data_source->apply_selection_filter(mask_memory);
  }

  /* 5. 应用行过滤器 */
  if (use_filters) {
    /* 构建列名称到列值的映射 */
    Map<StringRef, const ColumnValues *> columns;
    for (const ColumnLayout &column : spreadsheet_layout.columns) {
      columns.add(column.values->name(), column.values);
    }

    /* 遍历所有过滤器 */
    LISTBASE_FOREACH (const SpreadsheetRowFilter *, row_filter, &sspreadsheet.row_filters) {
      if (row_filter->flag & SPREADSHEET_ROW_FILTER_ENABLED) {
        if (!columns.contains(row_filter->column_name)) {
          continue;  /* 列不存在，跳过 */
        }
        mask = apply_row_filter(*row_filter, columns, mask, mask_memory);
      }
    }
  }

  return mask;
}
```

### 检查是否使用过滤器

```cpp
/* 来源: spreadsheet_row_filter.cc */
static bool use_row_filters(const SpaceSpreadsheet &sspreadsheet)
{
  /* 1. 检查过滤器是否启用 */
  if (!(sspreadsheet.filter_flag & SPREADSHEET_FILTER_ENABLE)) {
    return false;
  }

  /* 2. 检查是否有过滤器 */
  if (BLI_listbase_is_empty(&sspreadsheet.row_filters)) {
    return false;
  }

  return true;
}

static bool use_selection_filter(const SpaceSpreadsheet &sspreadsheet,
                                 const DataSource &data_source)
{
  /* 1. 检查选择过滤标志 */
  if (!(sspreadsheet.filter_flag & SPREADSHEET_FILTER_SELECTED_ONLY)) {
    return false;
  }

  /* 2. 检查数据源是否支持 */
  if (!data_source.has_selection_filter()) {
    return false;
  }

  return true;
}
```

## 过滤器操作实现

### 通用过滤模板

```cpp
/* 来源: spreadsheet_row_filter.cc */
template<typename T, typename OperationFn>
static IndexMask apply_filter_operation(const VArray<T> &data,
                                        OperationFn check_fn,
                                        const IndexMask &mask,
                                        IndexMaskMemory &memory)
{
  return IndexMask::from_predicate(
      mask, GrainSize(1024), memory, [&](const int64_t i) { return check_fn(data[i]); });
}
```

### 浮点数过滤

```cpp
/* 来源: spreadsheet_row_filter.cc */
static IndexMask apply_row_filter(const SpreadsheetRowFilter &row_filter,
                                  const Map<StringRef, const ColumnValues *> &columns,
                                  const IndexMask &prev_mask,
                                  IndexMaskMemory &memory)
{
  const ColumnValues &column = *columns.lookup(row_filter.column_name);
  const GVArray &column_data = column.data();

  /* 浮点数类型 */
  if (column_data.type().is<float>()) {
    const float value = row_filter.value_float;
    switch (row_filter.operation) {
      case SPREADSHEET_ROW_FILTER_EQUAL: {
        const float threshold = row_filter.threshold;
        return apply_filter_operation(
            column_data.typed<float>(),
            [&](const float cell) { return std::abs(cell - value) < threshold; },
            prev_mask,
            memory);
      }
      case SPREADSHEET_ROW_FILTER_GREATER: {
        return apply_filter_operation(
            column_data.typed<float>(),
            [&](const float cell) { return cell > value; },
            prev_mask,
            memory);
      }
      case SPREADSHEET_ROW_FILTER_LESS: {
        return apply_filter_operation(
            column_data.typed<float>(),
            [&](const float cell) { return cell < value; },
            prev_mask,
            memory);
      }
    }
  }

  /* ... 其他类型类似处理 */

  return prev_mask;  /* 不支持的类型，返回原掩码 */
}
```

```cpp
static void spreadsheet_filter_panel_draw(const bContext *C, Panel *panel)
{
  ui::Layout &layout = *panel->layout;
  SpaceSpreadsheet *sspreadsheet = CTX_wm_space_spreadsheet(C);
  PointerRNA *filter_ptr = ui::panel_custom_data_get(panel);
  SpreadsheetRowFilter *filter = (SpreadsheetRowFilter *)filter_ptr->data;

  /* 获取列信息 */
  const SpreadsheetColumn *column = lookup_visible_column_for_filter(*sspreadsheet,

  /* 1. 列选择器 */
  ui::Layout *row = &layout.row(true);
  row->prop(filter_ptr, "column_name", ui::ITEM_R_DEFAULT, "", ICON_NONE);

  /* 2. 操作符选择器 (根据数据类型) */
  if (column) {
    const eSpreadsheetColumnValueType data_type = (eSpreadsheetColumnValueType)column->data_type;
    row = &layout.row(true);

    /* 布尔和实例只有等于操作 */
    if (ELEM(data_type, SPREADSHEET_VALUE_TYPE_BOOL, SPREADSHEET_VALUE_TYPE_INSTANCES)) {
      row->prop(filter_ptr, "operation", ui::ITEM_R_DEFAULT, "", ICON_NONE);
    }
    else {
      row->prop(filter_ptr, "operation", ui::ITEM_R_DEFAULT, "", ICON_NONE);

  if (column) {
    const eSpreadsheetColumnValueType data_type = (eSpreadsheetColumnValueType)column->data_type;

    switch (data_type) {
      case SPREADSHEET_VALUE_TYPE_BOOL:
        row = &layout.row(true);
```

```cpp
/* 来源: spreadsheet_row_filter_ui.cc */
static void spreadsheet_filter_panel_draw_header(const bContext *C, Panel *panel)
{
  ui::Layout &layout = *panel->layout;
  SpaceSpreadsheet *sspreadsheet = CTX_wm_space_spreadsheet(C);
  PointerRNA *filter_ptr = ui::panel_custom_data_get(panel);
  const SpreadsheetRowFilter *filter = (SpreadsheetRowFilter *)filter_ptr->data;

  /* 查找列 */
  const SpreadsheetColumn *column = lookup_visible_column_for_filter(*sspreadsheet,
                                                                     filter->column_name);

  /* 检查是否启用和列是否存在 */
  if (!(sspreadsheet->filter_flag & SPREADSHEET_FILTER_ENABLE) ||
      (column == nullptr && !filter->column_name[0]))
  {
    layout.active_set(false);
  }

  /* 启用复选框 */
  ui::Layout *row = &layout.row(true);
  row->emboss_set(ui::EmbossType::None);
  row->prop(filter_ptr, "enabled", ui::ITEM_R_ICON_ONLY, "", ICON_NONE);

  /* 显示过滤器描述 */
  if (filter->column_name[0] == '\0') {
    row->label(IFACE_("Filter"), ICON_NONE);
  }
  else if (column == nullptr) {
    row->label(filter->column_name, ICON_NONE);
  }
  else {
    const eSpreadsheetColumnValueType data_type = (eSpreadsheetColumnValueType)column->data_type;
    std::stringstream ss;
    ss << filter->column_name;
    ss << " ";
    ss << operation_string(data_type, (eSpreadsheetFilterOperation)filter->operation);
    ss << " ";
    ss << value_string(*filter, data_type);
    row->label(ss.str(), ICON_NONE);
  }

  /* 删除按钮 */
  row = &layout.row(true);
  row->emboss_set(ui::EmbossType::None);
  const int current_index = BLI_findindex(&sspreadsheet->row_filters, filter);
  PointerRNA op_ptr = row->op("SPREADSHEET_OT_remove_row_filter_rule", "", ICON_X);
  RNA_int_set(&op_ptr, "index", current_index);
}
```

### 操作符字符串生成

```cpp
/* 来源: spreadsheet_row_filter_ui.cc */
static std::string operation_string(const eSpreadsheetColumnValueType data_type,
                                    const eSpreadsheetFilterOperation operation)
{
  if (ELEM(data_type, SPREADSHEET_VALUE_TYPE_BOOL, SPREADSHEET_VALUE_TYPE_INSTANCES)) {
    return "=";
  }

  switch (operation) {
    case SPREADSHEET_ROW_FILTER_EQUAL:
      return "=";
    case SPREADSHEET_ROW_FILTER_GREATER:
      return ">";
    case SPREADSHEET_ROW_FILTER_LESS:
      return "<";
  }
  BLI_assert_unreachable();
  return "";
}
```

### 值字符串格式化

```cpp
/* 来源: spreadsheet_row_filter_ui.cc */
static std::string value_string(const SpreadsheetRowFilter &row_filter,
                                const eSpreadsheetColumnValueType data_type)
{
  switch (data_type) {
    case SPREADSHEET_VALUE_TYPE_INT8:
    case SPREADSHEET_VALUE_TYPE_INT32:
    case SPREADSHEET_VALUE_TYPE_INT64:
      return std::to_string(row_filter.value_int);

    case SPREADSHEET_VALUE_TYPE_FLOAT: {
      std::ostringstream result;
      result.precision(3);
      result << std::fixed << row_filter.value_float;
      return result.str();
    }

    case SPREADSHEET_VALUE_TYPE_INT32_2D: {
      return fmt::format("({}, {})",
                         row_filter.value_int2[0],
                         row_filter.value_int2[1]);
    }

    case SPREADSHEET_VALUE_TYPE_INT32_3D: {
      return fmt::format("({}, {}, {})",
                         row_filter.value_int3[0],
                         row_filter.value_int3[1],
                         row_filter.value_int3[2]);
    }

    case SPREADSHEET_VALUE_TYPE_FLOAT2: {
      std::ostringstream result;
      result.precision(3);
      result << std::fixed << "(" << row_filter.value_float2[0] << ", "
             << row_filter.value_float2[1] << ")";
      return result.str();
    }

    case SPREADSHEET_VALUE_TYPE_FLOAT3: {
      std::ostringstream result;
      result.precision(3);
      result << std::fixed << "(" << row_filter.value_float3[0] << ", "
             << row_filter.value_float3[1] << ", " << row_filter.value_float3[2] << ")";
      return result.str();
    }

    case SPREADSHEET_VALUE_TYPE_BOOL:
      return (row_filter.flag & SPREADSHEET_ROW_FILTER_BOOL_VALUE) ? "True" : "False";

    case SPREADSHEET_VALUE_TYPE_INSTANCES:
      return row_filter.value_string ? row_filter.value_string : "";

    case SPREADSHEET_VALUE_TYPE_COLOR:
    case SPREADSHEET_VALUE_TYPE_BYTE_COLOR: {
      std::ostringstream result;
      result.precision(3);
      result << std::fixed << "(" << row_filter.value_color[0] << ", "
             << row_filter.value_color[1] << ", " << row_filter.value_color[2] << ", "
             << row_filter.value_color[3] << ")";
      return result.str();
    }

    case SPREADSHEET_VALUE_TYPE_STRING:
      return row_filter.value_string;

    default:
      return "";
  }
}
```

```cpp
/* 来源: spreadsheet_ops.cc */
static wmOperatorStatus row_filter_add_exec(bContext *C, wmOperator * /*op*/)
{
  SpaceSpreadsheet *sspreadsheet = CTX_wm_space_spreadsheet(C);

  /* 创建新过滤器 */
  SpreadsheetRowFilter *row_filter = spreadsheet_row_filter_new();

  /* 添加到链表 */
  BLI_addtail(&sspreadsheet->row_filters, row_filter);

  /* 通知更新 */
  WM_event_add_notifier(C, NC_SPACE | ND_SPACE_SPREADSHEET, sspreadsheet);

  return OPERATOR_FINISHED;
}
```

### 移除过滤器

```cpp
/* 来源: spreadsheet_ops.cc */
static wmOperatorStatus row_filter_remove_exec(bContext *C, wmOperator *op)
{
  SpaceSpreadsheet *sspreadsheet = CTX_wm_space_spreadsheet(C);

  /* 查找过滤器 */
  SpreadsheetRowFilter *row_filter = (SpreadsheetRowFilter *)BLI_findlink(
      &sspreadsheet->row_filters, RNA_int_get(op->ptr, "index"));

  if (row_filter == nullptr) {
    return OPERATOR_CANCELLED;
  }

  /* 从链表移除 */
  BLI_remlink(&sspreadsheet->row_filters, row_filter);

  /* 释放 */
  spreadsheet_row_filter_free(row_filter);

  /* 通知更新 */
  WM_event_add_notifier(C, NC_SPACE | ND_SPACE_SPREADSHEET, sspreadsheet);

  return OPERATOR_FINISHED;
}

static void SPREADSHEET_OT_remove_row_filter_rule(wmOperatorType *ot)
{

  ot->description = "Remove a row filter from the rules";

  ot->poll = ED_operator_spreadsheet_active;
```

## 完整过滤流程示例

```cpp
/* 假设场景:
   - 1000 行数据
   - 3 个过滤器: Position.x > 5.0, Position.y < 10.0, Index == 100
*/

/* 步骤 1: 初始掩码 (所有行) */
IndexMask mask(1000);  // [0, 1, 2, ..., 999]

/* 步骤 2: 应用第一个过滤器 (Position.x > 5.0) */
mask = apply_filter_operation(
    position_x_array,
    [](float x) { return x > 5.0f; },
    mask,
    memory);
// mask 现在包含: [10, 11, 15, 20, ...] (只有满足条件的)

/* 步骤 3: 应用第二个过滤器 (Position.y < 10.0) */
mask = apply_filter_operation(
    position_y_array,
    [](float y) { return y < 10.0f; },
    mask,
    memory);
// mask 现在包含: [10, 15, ...] (同时满足两个条件)

/* 步骤 4: 应用第三个过滤器 (Index == 100) */
mask = apply_filter_operation(
    index_array,
    [](int idx) { return idx == 100; },
    mask,
    memory);
// mask 现在包含: [100] (如果 100 在之前的掩码中)

/* 步骤 5: 使用过滤后的掩码进行渲染 */
for (int64_t i : mask) {
  draw_row(i);
}
```

## 总结

行过滤系统的关键特点：

1. **类型安全**: 支持所有电子表格数据类型
2. **高效**: 使用索引掩码和并行处理
3. **灵活**: 支持多种比较操作 (等于、大于、小于)
4. **用户友好**: 直观的 UI 和实时预览
5. **可组合**: 多个过滤器可以叠加使用

**支持的数据类型**:
- 基本类型: bool, int8, int32, int64, float
- 向量: float2, float3, int2, int3
- 颜色: ColorGeometry4f, ColorGeometry4b
- 字符串: std::string
- 特殊: InstanceReference

**操作类型**:
- `EQUAL`: 相等比较 (浮点数支持阈值)
- `GREATER`: 大于比较
- `LESS`: 小于比较
