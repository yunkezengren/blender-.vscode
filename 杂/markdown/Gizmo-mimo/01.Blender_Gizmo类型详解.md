## 概述

Blender 的 Gizmo 系统是一个强大的交互框架，允许用户通过直观的 3D/2D 控件来操作对象、参数和数据。Gizmo 分为两大类：

1. **核心库 Gizmos** - 可重用的基础 gizmo 类型
2. **专用 Gizmo Groups** - 针对特定编辑器和功能的 gizmo 集合

<span style="background-color: #2E7D32; color: white; padding: 2px 6px; border-radius: 3px;">★ Insight</span>
所有 gizmo 类型都在 `source/blender/editors/gizmo_library/gizmo_types/` 目录中定义，并通过 `ED_gizmotypes_*()` 函数注册。

### 2.1. 3D 箭头 Gizmo (GIZMO_GT_arrow_3d)

**定义位置**: `source/blender/editors/gizmo_library/gizmo_types/arrow3d_gizmo.cc:594-597`

```cpp
void ED_gizmotypes_arrow_3d()
{
  WM_gizmotype_append(GIZMO_GT_arrow_3d);
}
```

**功能**:
- 用于单轴移动或值调整的箭头形 gizmo
- 支持多种箭头样式

**样式 (draw_style)**:
```cpp
enum {
  ED_GIZMO_ARROW_STYLE_NORMAL = 0,   // 标准箭头
  ED_GIZMO_ARROW_STYLE_CROSS = 1,    // 十字形
  ED_GIZMO_ARROW_STYLE_BOX = 2,      // 盒形
  ED_GIZMO_ARROW_STYLE_CONE = 3,     // 圆锥形
  ED_GIZMO_ARROW_STYLE_PLANE = 4,    // 平面形
};
```

**变换标志 (transform)**:
```cpp
enum {
  ED_GIZMO_ARROW_XFORM_FLAG_INVERTED = (1 << 3),  // 反向偏移
  ED_GIZMO_ARROW_XFORM_FLAG_CONSTRAINED = (1 << 4), // 约束到属性宽度
};
```

**绘制选项 (draw_options)**:
```cpp
enum {
  ED_GIZMO_ARROW_DRAW_FLAG_STEM = (1 << 0),   // 显示箭杆
  ED_GIZMO_ARROW_DRAW_FLAG_ORIGIN = (1 << 1), // 显示原点
};
```

**使用示例**:
```cpp
// 创建箭头 gizmo
wmGizmo *gz = WM_gizmo_new("GIZMO_GT_arrow_3d", gzgroup, nullptr);

// 设置样式
RNA_enum_set(gz->ptr, "draw_style", ED_GIZMO_ARROW_STYLE_BOX);

// 绑定到属性
WM_gizmo_target_property_def_rna(gz, "offset", &ptr, "energy", 0);
```

**定义位置**: `source/blender/editors/gizmo_library/gizmo_types/arrow3d_gizmo.cc:594-597`

## 总结

Blender 的 gizmo 系统提供了丰富的交互控件：

1. **核心库** (10 种): 箭头、拨盘、移动、边界框(2D/3D)、按钮、原始形状、空白、捕捉、网格预选择
2. **专用 Groups**: 节点编辑器(7种)、3D视图(10+种)、变换系统
3. **总计**: 20+ 种不同的 gizmo 类型和组合

每个 gizmo 类型都有明确的：
- **样式选项** (draw_style)
- **变换标志** (transform)
- **绘制选项** (draw_options)
- **部件定义** (parts)

<span style="background-color: #1976D2; color: white; padding: 2px 6px; border-radius: 3px;">★ Key Point</span>
所有 gizmo 类型都遵循统一的架构：通过 `WM_gizmotype_append()` 注册，使用 RNA 属性配置，通过回调函数实现交互逻辑。
