## 1. 概述

Gizmo 系统通过修饰键（Modifier Keys）提供增强的交互功能，主要包括：

- **精确模式**（Shift）：降低灵敏度，实现微调
- **捕捉模式**（Ctrl）：启用网格、边缘或其他元素的捕捉
- **临时功能切换**：改变 gizmo 的行为（如等比缩放）

<mark style="background-color: #2E7D32; color: white; padding: 2px 6px; border-radius: 3px;">★ Insight</mark>
修饰键处理分为两个层次：**全局系统**（由窗口管理器处理）和**局部实现**（由具体 gizmo 处理）。

## 3. Cage2d Gizmo 的修饰键实现

### 3.1. Shift - 等比缩放

Cage2d gizmo 对 Shift 键有**特殊处理**：它不使用标准的精确模式，而是用于**临时启用等比缩放**。

**源码位置**: `source/blender/editors/gizmo_library/gizmo_types/cage2d_gizmo.cc:1164-1180`

```cpp
static wmOperatorStatus gizmo_cage2d_modal(bContext *C,
                                           wmGizmo *gz,
                                           const wmEvent *event,
                                           eWM_GizmoFlagTweak /*tweak_flag*/)
{
  RectTransformInteraction *data = static_cast<RectTransformInteraction *>(gz->interaction_data);
  int transform_flag = RNA_enum_get(gz->ptr, \"transform\");

  // 如果未在定义中启用等比缩放
  if ((transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM) == 0) {
    /* WARNING: Checking the events modifier only makes sense as long as `tweak_flag`
     * remains unused (this controls #WM_GIZMO_TWEAK_PRECISE by default). */
    const bool use_temp_uniform = (event->modifier & KM_SHIFT) != 0;
    const bool changed = data->use_temp_uniform != use_temp_uniform;
    data->use_temp_uniform = use_temp_uniform;

    if (use_temp_uniform) {
      transform_flag |= ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM;
    }

    if (changed) {
      /* Always refresh. */
    }
    else if (event->type != MOUSEMOVE) {
      return OPERATOR_RUNNING_MODAL;
    }
  }
  // ... 后续处理
}
```

**工作流程**：
1. 检查 `event->modifier & KM_SHIFT`
2. 设置 `data->use_temp_uniform = true`
3. 临时添加 `SCALE_UNIFORM` 标志到 `transform_flag`
4. 后续的缩放计算使用等比模式

### 3.2. 实现细节

#### 3.2.1. 临时标志存储

```cpp
struct RectTransformInteraction {
  float orig_mouse[2];
  float orig_matrix_offset[4][4];
  float orig_matrix_final_no_offset[4][4];
  Dial *dial;
  bool use_temp_uniform;  // ← 用于存储临时等比状态
};
```

#### 3.2.2. 动态标志获取

```cpp
static int gizmo_cage2d_transform_flag_get(const wmGizmo *gz)
{
  RectTransformInteraction *data = static_cast<RectTransformInteraction *>(gz->interaction_data);
  int transform_flag = RNA_enum_get(gz->ptr, \"transform\");

  if (data) {
    if (data->use_temp_uniform) {
      transform_flag |= ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM;
    }
  }

  return transform_flag;
}
```

**使用场景**：
- 在绘制时确定是否显示等比提示
- 在缩放计算时决定是否强制相同比例

#### 3.2.3. 缩放计算中的应用

```cpp
// 在 gizmo_cage2d_modal 中
if (transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM) {
  if (constrain_axis[0] == false && constrain_axis[1] == false) {
    if (draw_style == ED_GIZMO_CAGE2D_STYLE_CIRCLE) {
      scale[1] = scale[0] = len_v2(scale);  // 圆形：保持半径
    }
    else {
      scale[1] = scale[0] = (scale[1] + scale[0]) / 2.0f;  // 平均值
    }
  }
  else if (constrain_axis[0] == false) {
    scale[1] = scale[0];  // X 轴缩放影响 Y
  }
  else if (constrain_axis[1] == false) {
    scale[0] = scale[1];  // Y 轴缩放影响 X
  }
}
```

### 4.2. 模态操作流程

#### 4.2.1. 整体流程图


#### 4.2.2. 详细执行路径

```cpp
// 1. 用户按下 Shift
// 2. 窗口系统捕获事件
// 3. 调用 gizmo_tweak_modal

static wmOperatorStatus gizmo_tweak_modal(bContext *C, wmOperator *op, const wmEvent *event)
{
  GizmoTweakData *mtweak = static_cast<GizmoTweakData *>(op->customdata);
  wmGizmo *gz = mtweak->gz_modal;

  // 3.1. 处理模态映射事件
  if (event->type == EVT_MODAL_MAP) {
    switch (event->val) {
      case TWEAK_MODAL_PRECISION_ON:
        mtweak->flag |= WM_GIZMO_TWEAK_PRECISE;  // ← 设置标志
        break;
      case TWEAK_MODAL_SNAP_ON:
        mtweak->flag |= WM_GIZMO_TWEAK_SNAP;
        break;
      case TWEAK_MODAL_PRECISION_OFF:
        mtweak->flag &= ~WM_GIZMO_TWEAK_PRECISE;
        break;
      case TWEAK_MODAL_SNAP_OFF:
        mtweak->flag &= ~WM_GIZMO_TWEAK_SNAP;
        break;
    }
  }

  // 3.2. 调用 gizmo 的 modal 回调
  wmGizmoFnModal modal_fn = gz->custom_modal ? gz->custom_modal : gz->type->modal;
  if (modal_fn) {
    // 传递标志给 gizmo
    const wmOperatorStatus modal_retval = modal_fn(C, gz, event,
                                                   eWM_GizmoFlagTweak(mtweak->flag));
  }

  return OPERATOR_PASS_THROUGH;
}
```

#### 4.3.1. 传递路径

#### 4.3.2. gizmo 如何使用标志

**标准用法**（检查 tweak_flag）：

```cpp
static wmOperatorStatus my_gizmo_modal(bContext *C,
                                       wmGizmo *gz,
                                       const wmEvent *event,
                                       eWM_GizmoFlagTweak tweak_flag)
{
  // 检查精确模式
  bool use_precision = (tweak_flag & WM_GIZMO_TWEAK_PRECISE) != 0;
  float sensitivity = use_precision ? 0.01f : 0.1f;

  // 检查捕捉模式
  bool use_snap = (tweak_flag & WM_GIZMO_TWEAK_SNAP) != 0;
  if (use_snap) {
    // 应用捕捉逻辑
    value = snap_to_grid(value);
  }

  // 使用灵敏度计算变换
  // ...
}
```

**Cage2d 的特殊用法**（直接检查事件）：

```cpp
static wmOperatorStatus gizmo_cage2d_modal(bContext *C,
                                           wmGizmo *gz,
                                           const wmEvent *event,
                                           eWM_GizmoFlagTweak /*tweak_flag*/)
{
  // 注意：tweak_flag 参数被注释掉，未使用
  // 而是直接检查 event->modifier
```

**原因**：
- `tweak_flag` 默认用于精确/捕捉模式

### 5.2. 移动 Gizmo

**源码**: `source/blender/editors/gizmo_library/gizmo_types/move3d_gizmo.cc`

```cpp
static wmOperatorStatus gizmo_move_modal(bContext *C,
                                         wmGizmo *gz,
                                         const wmEvent *event,
                                         eWM_GizmoFlagTweak tweak_flag)
{
  // 标准修饰键处理
  float sensitivity = 1.0f;
  if (tweak_flag & WM_GIZMO_TWEAK_PRECISE) {
    sensitivity = 0.1f;
  }

  // 计算移动
  float delta[3];
  sub_v3_v3v3(delta, point_local, data->orig_mouse);
  mul_v3_fl(delta, sensitivity);

  // 应用捕捉
  if (tweak_flag & WM_GIZMO_TWEAK_SNAP) {
    // 捕捉到网格或其他元素
    ED_gizmo_snap_apply(delta, snap_context);
  }

  // 更新位置
  add_v3_v3(gz->matrix_offset[3], delta);
}
```

## 6. 自定义修饰键处理

### 6.1. 直接事件检查（推荐用于特殊需求）

当标准的精确/捕捉模式不满足需求时，可以直接检查事件：

```cpp
static wmOperatorStatus my_gizmo_modal(bContext *C,
                                       wmGizmo *gz,
                                       const wmEvent *event,
                                       eWM_GizmoFlagTweak tweak_flag)
{
  MyGizmoData *data = (MyGizmoData *)gz->interaction_data;

  // 1. 检查 Shift - 特殊功能（如等比）
  bool shift_down = (event->modifier & KM_SHIFT) != 0;
  if (shift_down != data->shift_was_down) {
    // Shift 状态改变，触发特殊逻辑
    data->use_special_mode = shift_down;
    // 触发刷新
  }

  // 2. 检查 Ctrl - 捕捉
  bool ctrl_down = (event->modifier & KM_CTRL) != 0;

  // 3. 检查 Alt - 其他功能
  bool alt_down = (event->modifier & KM_ALT) != 0;

  // 4. 组合键
  if (shift_down && ctrl_down) {
    // Shift+Ctrl 组合功能
    // 例如：中心缩放
  }

  // 5. 处理变换
  if (data->use_special_mode) {
    // 特殊模式逻辑
  }
  else {
    // 标准模式逻辑
  }

  return OPERATOR_RUNNING_MODAL;
}
```

### 6.2. 使用 tweak_flag（推荐用于标准功能）

## 7. Cage2d 修饰键完整示例

### 7.1. 完整的 Shift 处理流程

```cpp
// 文件: source/blender/editors/gizmo_library/gizmo_types/cage2d_gizmo.cc

// 步骤 1: 在 modal 中检测 Shift
static wmOperatorStatus gizmo_cage2d_modal(/*...*/)
{
  RectTransformInteraction *data = /*... */;
  int transform_flag = RNA_enum_get(gz->ptr, \"transform\");

  // 如果未在定义中启用等比缩放
  if ((transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM) == 0) {
    // 直接检查事件修饰符
    const bool use_temp_uniform = (event->modifier & KM_SHIFT) != 0;
    const bool changed = data->use_temp_uniform != use_temp_uniform;

    data->use_temp_uniform = use_temp_uniform;

    if (use_temp_uniform) {
      transform_flag |= ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM;
    }

    if (changed) {
      // 标记需要刷新
    }
  }

  // 步骤 2: 在缩放计算中使用
  if (transform_flag & ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM) {
    // 强制等比
    if (constrain_axis[0] == false && constrain_axis[1] == false) {
      scale[1] = scale[0] = (scale[1] + scale[0]) / 2.0f;
    }
  }

  // 步骤 3: 在绘制中反映
  // gizmo_cage2d_transform_flag_get 会包含临时标志
}

// 步骤 4: 获取当前标志（用于绘制和选择）
static int gizmo_cage2d_transform_flag_get(const wmGizmo *gz)
{
  RectTransformInteraction *data = /*... */;
  int transform_flag = RNA_enum_get(gz->ptr, \"transform\");

  if (data && data->use_temp_uniform) {
    transform_flag |= ED_GIZMO_CAGE_XFORM_FLAG_SCALE_UNIFORM;
  }

  return transform_flag;
}

// 步骤 5: 退出时清理
static void gizmo_cage2d_exit(bContext *C, wmGizmo *gz, const bool cancel)
{
  RectTransformInteraction *data = /*... */;

  // 交互数据会被释放，临时标志自动清除
  if (data->dial) {
    BLI_dial_free(data->dial);
  }
  MEM_freeN(data);
}
```

### 7.2. 与标准系统的对比

| 特性 | 标准系统 (Shift=精确) | Cage2d (Shift=等比) |
|------|----------------------|---------------------|
| **标志来源** | `tweak_flag` 参数 | `event->modifier` 直接检查 |
| **标志存储** | `mtweak->flag` | `data->use_temp_uniform` |
| **处理位置** | `gizmo_tweak_modal` | `gizmo_cage2d_modal` |
| **影响范围** | 通用灵敏度调整 | 特定变换模式切换 |
| **优点** | 统一、可配置 | 灵活、特定需求 |

## 8. 总结

### 8.1. 修饰键处理层次

```
层次 1: 键映射 (wm_gizmo_group.cc)
  ↓ 转换为模态事件
层次 2: 模态操作 (gizmo_tweak_modal)
  ↓ 设置标志
层次 3: Gizmo 回调 (gizmo_modal)
  ↓ 使用标志或直接检查
层次 4: 具体实现 (变换计算)
```

### 8.2. 最佳实践

#### ✅ 推荐做法

1. **使用标准系统**：优先使用 `tweak_flag` 参数
2. **直接检查**：特殊需求时检查 `event->modifier`
3. **临时存储**：在交互数据中保存修饰键状态
4. **动态获取**：使用辅助函数获取当前标志

#### ❌ 避免

1. **硬编码键值**：不要直接检查 `event->type == EVT_LEFTSHIFTKEY`
2. **忽略清理**：确保交互数据正确释放
3. **混淆标志**：不要同时使用 `tweak_flag` 和直接检查（除非有明确理由）

### 8.3. Cage2d 特殊性

Cage2d 选择直接检查事件的原因：
- **需求不同**：需要等比缩放而非精确模式
- **历史原因**：早期实现方式
- **灵活性**：可以独立于标准系统

### 8.4. 扩展指南

如果要为 Cage2d 添加 Ctrl 捕捉：

```cpp
// 在 modal 中添加
bool ctrl_down = (event->modifier & KM_CTRL) != 0;
if (ctrl_down) {
  // 应用捕捉到 point_local
  point_local[0] = round(point_local[0] / grid_size) * grid_size;
  point_local[1] = round(point_local[1] / grid_size) * grid_size;
}
```
