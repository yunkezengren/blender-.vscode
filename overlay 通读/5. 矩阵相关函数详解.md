# 5. 矩阵相关函数详解

## 5.1 矩阵系统概述

### 5.1.1 Blender中矩阵的作用和重要性

在Blender的渲染系统中，矩阵是3D图形变换的核心数学工具，负责实现以下关键功能：

- **坐标空间转换**：在不同坐标系之间转换顶点位置（局部空间→世界空间→视图空间→裁剪空间）
- **对象变换**：实现平移、旋转、缩放等基本变换操作
- **相机投影**：将3D场景投影到2D屏幕空间
- **视锥剔除**：优化渲染性能，剔除视野外的对象

### 5.1.2 矩阵在渲染管线中的位置

```cpp
// 渲染管线中的矩阵变换流程
局部坐标 → [模型矩阵] → 世界坐标 → [视图矩阵] → 视图坐标 → [投影矩阵] → 裁剪坐标 → NDC → 屏幕坐标
```

### 5.1.3 不同类型矩阵的用途

| 矩阵类型 | 用途 | 数据结构 |
|---------|------|----------|
| **模型矩阵** | 对象从局部空间到世界空间的变换 | `float4x4 model` |
| **视图矩阵** | 相机从世界空间到视图空间的变换 | `float4x4 viewmat` |
| **投影矩阵** | 3D到2D的投影变换 | `float4x4 winmat` |
| **法线矩阵** | 法线向量的正确变换 | `float3x3 normal_matrix` |

```cpp
void View::sync(const float4x4 &view_mat, const float4x4 &win_mat, int view_id)
{
  data_[view_id].viewmat = view_mat;           // 设置视图矩阵
  data_[view_id].viewinv = math::invert(view_mat); // 计算视图逆矩阵
  data_[view_id].winmat = win_mat;            // 设置投影矩阵
  data_[view_id].wininv = math::invert(win_mat); // 计算投影逆矩阵

  is_inverted_ = (is_negative_m4(view_mat.ptr()) == is_negative_m4(win_mat.ptr()));

  frustum_boundbox_calc(view_id);      // 计算视锥包围盒
  frustum_culling_planes_calc(view_id); // 计算剔除平面
  frustum_culling_sphere_calc(view_id); // 计算剔除球体

  dirty_ = true;
  manager_fingerprint_ = 0;
  sync_counter_ = (global_sync_counter_ += 2);
}
```

**功能分析**：
- 同步视图和投影矩阵到GPU缓冲区
- 自动计算逆矩阵，用于反向变换
- 更新视锥剔除数据，优化渲染性能
- 维护同步计数器，支持缓存失效检测

### 5.4.2 矩阵变换函数

#### 视锥包围盒计算
**文件位置**: `source/blender/draw/intern/draw_view.cc:46-90`

```cpp
void View::frustum_boundbox_calc(int view_id)
{
  MutableSpan<float4> corners = {culling_[view_id].frustum_corners.corners,
                                 int64_t(ARRAY_SIZE(culling_[view_id].frustum_corners.corners))};

  float left, right, bottom, top, near, far;
  bool is_persp = data_[view_id].winmat[3][3] == 0.0f;

  // 从投影矩阵提取视锥参数
  projmat_dimensions(data_[view_id].winmat.ptr(), &left, &right, &bottom, &top, &near, &far);

  // 设置近平面角点
  corners[0][2] = corners[3][2] = corners[7][2] = corners[4][2] = -near;
  corners[0][0] = corners[3][0] = left;
  corners[4][0] = corners[7][0] = right;
  corners[0][1] = corners[4][1] = bottom;
  corners[7][1] = corners[3][1] = top;

  // 计算远平面角点（考虑透视缩放）
  if (is_persp) {
    float sca_far = far / near;
    left *= sca_far;
    right *= sca_far;
    bottom *= sca_far;
    top *= sca_far;
  }

  // 设置远平面角点
  corners[1][2] = corners[2][2] = corners[6][2] = corners[5][2] = -far;
  corners[1][0] = corners[2][0] = left;
  corners[6][0] = corners[5][0] = right;
  corners[1][1] = corners[5][1] = bottom;
  corners[2][1] = corners[6][1] = top;

  // 转换到世界空间
  const float4x4 &view_inv = data_[view_id].viewinv;
  for (float4 &corner : corners) {
    corner = float4(math::transform_point(view_inv, float3(corner)), 1.0);
  }
}
```

#### 视锥剔除平面计算
**文件位置**: `source/blender/draw/intern/draw_view.cc:92-109`

```cpp
void View::frustum_culling_planes_calc(int view_id)
{
  float4x4 persmat = data_[view_id].winmat * data_[view_id].viewmat;
  
  // 从透视投影矩阵提取6个剔除平面
  planes_from_projmat(persmat.ptr(),
                      culling_[view_id].frustum_planes.planes[0], // 左平面
                      culling_[view_id].frustum_planes.planes[5], // 远平面
                      culling_[view_id].frustum_planes.planes[1], // 右平面
                      culling_[view_id].frustum_planes.planes[3], // 上平面
                      culling_[view_id].frustum_planes.planes[4], // 近平面
                      culling_[view_id].frustum_planes.planes[2]); // 下平面
  
  // 归一化平面方程
  for (float4 &plane : culling_[view_id].frustum_planes.planes) {
    float len = math::length(plane.xyz());
    if (len != 0.0f) {
      plane /= len;
    }
  }
}
```

**文件位置**: `source/blender/draw/intern/draw_view.hh:106-122`

```cpp
float far_clip(int view_id = 0) const
{
  BLI_assert(view_id < view_len_);
  if (is_persp(view_id)) {
    // 透视投影的远裁剪面计算
    return -data_[view_id].winmat[3][2] / (data_[view_id].winmat[2][2] + 1.0f);
  }
  // 正交投影的远裁剪面计算
  return -(data_[view_id].winmat[3][2] - 1.0f) / data_[view_id].winmat[2][2];
}

float near_clip(int view_id = 0) const
{
  BLI_assert(view_id < view_len_);
  if (is_persp(view_id)) {
    // 透视投影的近裁剪面计算
    return -data_[view_id].winmat[3][2] / (data_[view_id].winmat[2][2] - 1.0f);
  }
  // 正交投影的近裁剪面计算
  return -(data_[view_id].winmat[3][2] + 1.0f) / data_[view_id].winmat[2][2];
}
```

```cpp
inline ResourceHandleRange Manager::resource_handle(const ObjectRef &ref, float inflate_bounds)
{
  bool is_active_object = ref.is_active(object_active);
  bool is_active_edit_mode = object_active &&
                             (DRW_object_is_in_edit_mode(object_active) ||
                              ELEM(object_active->mode, OB_MODE_TEXTURE_PAINT, OB_MODE_SCULPT)) &&
                             ref.object->mode == object_active->mode;
  
  if (ref.duplis_) {
    // 处理重复对象
    uint start = resource_len_;
    for (const DupliObject *dupli : *ref.duplis_) {
      // 同步每个重复对象的变换矩阵
      matrix_buf.current().get_or_resize(resource_len_).sync(float4x4(dupli->mat));
      bounds_buf.current().get_or_resize(resource_len_) = proto_bounds;
      resource_len_++;
    }
    return ResourceHandleRange(ResourceHandle(start, (ref.object->transflag & OB_NEG_SCALE) != 0),
                           resource_len_ - start);
  }
  else {
    // 同步单个对象的变换矩阵
    matrix_buf.current().get_or_resize(resource_len_).sync(*ref.object);
    bounds_buf.current().get_or_resize(resource_len_).sync(*ref.object, inflate_bounds);
    infos_buf.current()
        .get_or_resize(resource_len_)
        .sync(ref, is_active_object, is_active_edit_mode);
    return ResourceHandle(resource_len_++, (ref.object->transflag & OB_NEG_SCALE) != 0);
  }
}
```

```cpp
float4x4 winmat_polygon_offset(const float4x4 &winmat, float offset)
{
  float view_dist = dist;
  /* 正交相机特殊处理 */
  if (persp == RV3D_CAMOB && is_persp == false) {
```

```cpp
void View::compute_visibility(ObjectBoundsBuf &bounds,
                          ObjectInfosBuf &infos,
                          uint resource_len,
                          bool debug_freeze)
{
  // 计算可见性字长
  uint word_per_draw = this->visibility_word_per_draw();
  uint words_len = (view_len_ == 1) ? divide_ceil_u(resource_len, 32) :
                                      resource_len * word_per_draw;
  words_len = ceil_to_multiple_u(max_ii(1, words_len), 4);
  
  // 调整可见性缓冲区大小
  visibility_buf_.resize(words_len);
  
  // 使用计算着色器进行GPU端剔除
  if (do_visibility_) {
    gpu::Shader *shader = DRW_shader_draw_visibility_compute_get();
    GPU_shader_bind(shader);
    GPU_shader_uniform_1i(shader, "resource_len", resource_len);
    GPU_shader_uniform_1i(shader, "view_len", view_len_);
    GPU_shader_uniform_1i(shader, "visibility_word_per_draw", word_per_draw);
    GPU_storagebuf_bind(bounds, GPU_shader_get_ssbo_binding(shader, "bounds_buf"));
    GPU_storagebuf_bind(visibility_buf_, GPU_shader_get_ssbo_binding(shader, "visibility_buf"));
    GPU_uniformbuf_bind(frozen_ ? data_freeze_ : data_, DRW_VIEW_UBO_SLOT);
    GPU_uniformbuf_bind(frozen_ ? culling_freeze_ : culling_, DRW_VIEW_CULLING_UBO_SLOT);
    GPU_compute_dispatch(shader, divide_ceil_u(resource_len, DRW_VISIBILITY_GROUP_SIZE), 1, 1);
    GPU_memory_barrier(GPU_BARRIER_SHADER_STORAGE);
  }
}
```
