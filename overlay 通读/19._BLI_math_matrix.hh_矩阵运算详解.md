## 矩阵运算概述

### 1.1 矩阵在3D图形学中的重要性

矩阵是3D图形学的核心数学工具，用于表示：
- **变换操作**：平移、旋转、缩放
- **投影变换**：透视投影、正交投影
- **坐标系转换**：世界坐标、相机坐标、屏幕坐标
- **复合变换**：多个变换的组合

### 1.2 Blender矩阵系统的设计理念

Blender的矩阵系统基于以下设计原则：

```cpp
// source/blender/blenlib/BLI_math_matrix_types.hh:12-19
/**
 * The `blender::MatBase<T, NumCol, NumRow>` is a Row x Col matrix (in mathematical notation) laid
 * out as column major in memory.
 * 
 * This class overloads `+, -, *` and `+=, -=, *=` mathematical operators.
 * They are all using per component operation, except for a few:
 * `MatBase<C,R> * Vector<C>` the vector product with the matrix.
 * `Vector<R> * MatBase<C,R>` the vector product with the **transposed** matrix.
 * `MatBase<C,R> * MatBase<R,C>` and `MatBase<C,R> *= MatBase<R,C>` the matrix multiplication.
 */
```

**核心特点**：
- **列主序存储**：与OpenGL/Vulkan等图形API兼容
- **模板化设计**：支持任意数值类型（float, double等）
- **运算符重载**：提供直观的数学运算语法
- **类型安全**：编译时检查矩阵维度匹配

## 核心函数详解

### 2.1 transform_point 函数族详解

#### 2.1.1 2D点变换（2x2矩阵）

```cpp
// source/blender/blenlib/BLI_math_matrix.hh:1629-1632
template<typename T>
VecBase<T, 2> transform_point(const MatBase<T, 2, 2> &mat, const VecBase<T, 2> &point)
{
  return mat * point;
}
```

**功能分析**：
- **参数**：
  - `mat`: 2x2变换矩阵
  - `point`: 2D点坐标
- **返回值**：变换后的2D点
- **数学原理**：简单的矩阵-向量乘法
- **适用场景**：纯线性变换（旋转、缩放、剪切）

**数学公式**：
```
[x']   [m00 m01] [x]
[y'] = [m10 m11] [y]
```

#### 2.1.2 2D点变换（3x3齐次矩阵）

```cpp
// source/blender/blenlib/BLI_math_matrix.hh:1635-1638
template<typename T>
VecBase<T, 2> transform_point(const MatBase<T, 3, 3> &mat, const VecBase<T, 2> &point)
{
  return mat.template view<2, 2>() * point + mat.location();
}
```

**功能分析**：
- **参数**：
  - `mat`: 3x3齐次坐标变换矩阵
  - `point`: 2D点坐标
- **返回值**：变换后的2D点
- **关键技术**：
  - `mat.template view<2, 2>()`: 提取左上角2x2子矩阵
  - `mat.location()`: 获取平移分量
- **数学原理**：仿射变换 = 线性变换 + 平移

**C++语法详解**：
```cpp
// template关键字的使用
mat.template view<2, 2>()  // 必须使用template关键字，因为view是模板成员函数
// 等价于：
auto linear_part = mat.view<2, 2>();  // 提取旋转/缩放部分
auto translation = mat.location();    // 提取平移部分
return linear_part * point + translation;
```

**数学公式**：
```
[x']   [m00 m01] [x]   [m02]
[y'] = [m10 m11] [y] + [m12]
[1 ]   [ 0   0 ] [1]   [ 1 ]
```

### 2.3 project_point 函数详解

```cpp
// source/blender/blenlib/BLI_math_matrix.hh:1665-1671
template<typename T, int N, int NumRow>
VecBase<T, N> project_point(const MatBase<T, N + 1, NumRow> &mat, const VecBase<T, N> &point)
{
  VecBase<T, N + 1> tmp(point, T(1));  // 齐次化
  tmp = mat * tmp;                      // 矩阵变换
  /* Absolute value to not flip the frustum upside down behind the camera. */
  return VecBase<T, N>(tmp) / math::abs(tmp[N]);  // 透视除法
}
```

**功能分析**：
- **透视投影**：3D坐标到2D屏幕坐标
- **关键步骤**：
  1. 齐次化：添加w=1分量
  2. 矩阵变换：应用投影矩阵
  3. 透视除法：除以w分量
- **技术细节**：使用`math::abs()`防止视锥体翻转

## C++语法和模板详解

### 3.1 模板语法深度解析

#### 3.1.1 函数模板基础

```cpp
template<typename T>
VecBase<T, 2> transform_point(const MatBase<T, 2, 2> &mat, const VecBase<T, 2> &point)
```

**语法解析**：
- `template<typename T>`: 声明模板参数T
- `T`: 可以是任意数值类型（float, double, int等）
- `VecBase<T, 2>`: 模板实例化，T为具体类型

**Python对比**：
```python
# Python的泛型实现（动态类型）
def transform_point(mat, point):
    return mat * point  # 运行时类型检查

# C++的模板实现（静态类型）
template<typename T>
VecBase<T, 2> transform_point(const MatBase<T, 2, 2> &mat, const VecBase<T, 2> &point)
{
    return mat * point;  // 编译时类型检查
}
```

#### 3.1.2 模板成员函数调用

```cpp
return mat.template view<3, 3>() * point + mat.location();
```

**语法详解**：
- `mat.template view<3, 3>()`: 模板成员函数调用
- `template`关键字：告诉编译器`view`是模板
- `<3, 3>`: 模板参数，指定子矩阵维度

**为什么需要template关键字**：
```cpp
template<typename T>
void func(const MatBase<T, 4, 4>& mat) {
    // 错误：编译器无法解析view<3, 3>
    // auto sub = mat.view<3, 3>();
    
    // 正确：使用template关键字
    auto sub = mat.template view<3, 3>();
}
```

**重载示例**：

### 5.2 SIMD优化实现

```cpp
// source/blender/blenlib/intern/math_matrix.cc:25-46
template<> float4x4 operator*(const float4x4 &a, const float4x4 &b)
{
#if BLI_HAVE_SSE2
  __m128 A0 = _mm_load_ps(a[0]);
  __m128 A1 = _mm_load_ps(a[1]);
  __m128 A2 = _mm_load_ps(a[2]);
  __m128 A3 = _mm_load_ps(a[3]);

  for (int i = 0; i < 4; i++) {
    __m128 B0 = _mm_set1_ps(b[i][0]);
    __m128 B1 = _mm_set1_ps(b[i][1]);
    __m128 B2 = _mm_set1_ps(b[i][2]);
    __m128 B3 = _mm_set1_ps(b[i][3]);

    __m128 sum = _mm_add_ps(_mm_add_ps(_mm_mul_ps(B0, A0), _mm_mul_ps(B1, A1)),
                            _mm_add_ps(_mm_mul_ps(B2, A2), _mm_mul_ps(B3, A3)));

    _mm_store_ps(result[i], sum);
  }
#else
  // 标量实现
#endif
}
```

**优化策略**：
- **SSE2指令集**：并行处理4个float
- **内存对齐**：确保16字节对齐访问
- **编译时选择**：根据CPU能力自动选择
