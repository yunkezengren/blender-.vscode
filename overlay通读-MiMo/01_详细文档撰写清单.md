**深度缓冲工作原理**:
```
渲染流程(当前问题):
1. 背景 → 深度缓冲
2. 物体 → 深度缓冲(遮挡信息)
3. 属性预览覆盖 → 不一定写入正确深度
4. 文本 → 可能被错误绘制在遮挡物前
