```python
# 原理: 采样深度缓冲，比较文本位置
def should_draw_text(text_3d_pos):

    text_depth = screen_pos[2]

    scene_depth = gpu.read_depth(screen_pos[0], screen_pos[1])

    # 如果文本在物体后面，不绘制
```

**核心思路**:
```glsl
// 着色器内核原理
void main()
{
    int vert_id = gl_VertexID;  // 0,1,2,3...
    int vector_idx = vert_id / 2;  // 哪个向量
    bool is_start = (vert_id & 1) == 0;  // 奇偶判断

    float3 pos = positions[vector_idx];
    float3 vector = vectors[vector_idx];

    if (is_start) {
        output_pos = pos;          // 起点
    } else {
        output_pos = pos + vector * scale;  // 终点
    }

    final_color = vector_color;
}
```

## 🎓 概念速查表

| 缩写 | 全称 | 作用 | 您的用途 |
|------|------|------|---------|
| UBO | Uniform Buffer Object | GPU全局参数 | 传视角矩阵 |
| VBO | Vertex Buffer Object | 顶点数据 | 传顶点+属性 |
| Shader | 着色器 | GPU程序 | 改变显示 |
| Pass | 渲染通道 | 绘制批次 | 管理绘制顺序 |
| Sub | 子通道 | 特定类型 | 管理mesh/my/curve |
| Batch | 绘制批次 | 一次完整绘制 | 直接提交GPU |
| Handle | 资源句柄 | 对象变换 | 让物体正确显示 |

**一句话记忆**:
```
Pass管批次 → Sub管类型 → Batch管具体绘制
Shader管视觉 → VBO管数据 → UBO管参数
```
