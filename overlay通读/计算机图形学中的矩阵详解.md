## 1. 概述  [[⬆](#目录)]

矩阵在计算机图形学中是表示线性变换的基本数学工具。它们可以表示旋转、缩放、平移、错切等几何变换，以及从3D世界到2D屏幕的投影变换。在3D图形管线中，矩阵用于将顶点坐标从一个坐标系转换到另一个坐标系。

Blender使用了一套完整的矩阵数学库，位于`source/blender/blenlib/`目录中，提供了高性能的矩阵运算功能。

## 3. 坐标空间  [[⬆](#目录)]

### 3.1. 局部空间  [[⬆](#目录)]

局部空间（Local Space）是物体自身的坐标系，原点通常位于物体的中心或某个特定参考点。

### 3.2. 世界空间  [[⬆](#目录)]

世界空间（World Space）是全局坐标系，所有物体都在这个统一的空间中定位。通过模型矩阵将局部坐标转换到世界坐标。

### 3.3. 视图空间  [[⬆](#目录)]

视图空间（View Space）也称为相机空间或眼空间，是以相机为原点的坐标系。通过视图矩阵将世界坐标转换到视图坐标。

### 3.4. 裁剪空间  [[⬆](#目录)]

裁剪空间（Clip Space）是经过投影变换后的坐标系，用于执行裁剪操作。在这个空间中，可见物体的坐标都在一个标准化的立方体内。

### 3.5. 屏幕空间  [[⬆](#目录)]

屏幕空间（Screen Space）是最终的2D坐标系，对应屏幕像素位置。通过透视除法和视口变换将裁剪坐标转换到屏幕坐标。

## 4. 变换操作  [[⬆](#目录)]

### 4.1. 平移变换  [[⬆](#目录)]

平移变换将物体沿指定方向移动。在齐次坐标中，平移通过4x4矩阵的最后一列实现。

**定义位置**: `source/blender/blenlib/BLI_math_matrix.hh:658-679`

```cpp
template<typename T, int NumCol, int NumRow, typename VectorT>
[[nodiscard]] MatBase<T, NumCol, NumRow> translate(const MatBase<T, NumCol, NumRow> &mat,
                                                   const VectorT &translation)
{
    using MatT = MatBase<T, NumCol, NumRow>;
    BLI_STATIC_ASSERT(VectorT::type_length <= MatT::col_len - 1,
                  "Translation should be at least 1 column less than the matrix.");
    constexpr int location_col = MatT::col_len - 1;
    
    MatT result = mat;
    unroll<VectorT::type_length>([&](auto c) {
        *reinterpret_cast<IntermediateVecT *>(
            &result[location_col]) += translation[c] *
                                  *reinterpret_cast<const IntermediateVecT *>(&mat[c]);
    });
    return result;
}
```

### 4.2. 旋转变换  [[⬆](#目录)]

旋转变换使物体绕指定轴旋转一定角度。Blender支持多种旋转表示方式：欧拉角、四元数、轴角等。

**定义位置**: `source/blender/blenlib/BLI_math_matrix.hh:681-716`

```cpp
template<typename T, int NumCol, int NumRow, typename AngleT>
[[nodiscard]] MatBase<T, NumCol, NumRow> rotate(const MatBase<T, NumCol, NumRow> &mat,
                                             const AxisAngleBase<T, AngleT> &rotation)
{
    using MatT = MatBase<T, NumCol, NumRow>;
    using Vec3T = typename MatT::vec3_type;
    const T angle_sin = sin(rotation.angle());
    const T angle_cos = cos(rotation.angle());
    const Vec3T &axis_vec = rotation.axis();

    MatT result = mat;
    // 针对主轴的优化实现
    if (axis_vec.x == T(1)) {
        unroll<MatT::row_len>([&](auto c) {
            result[2][c] = -angle_sin * mat[1][c] + angle_cos * mat[2][c];
            result[1][c] = angle_cos * mat[1][c] + angle_sin * mat[2][c];
        });
    }
    // ... 其他轴的优化处理
    return result;
}
```

### 4.3. 缩放变换  [[⬆](#目录)]

缩放变换改变物体的大小。均匀缩放在所有方向上等比例缩放，非均匀缩放在不同方向上有不同的缩放因子。

**定义位置**: `source/blender/blenlib/BLI_math_matrix.hh:718-727`

```cpp
template<typename T, int NumCol, int NumRow, typename VectorT>
[[nodiscard]] MatBase<T, NumCol, NumRow> scale(const MatBase<T, NumCol, NumRow> &mat,
                                              const VectorT &scale)
{
    BLI_STATIC_ASSERT(VectorT::type_length <= NumCol,
                  "Scale should be less or equal to the matrix in column count.");
    MatBase<T, NumCol, NumRow> result = mat;
    unroll<VectorT::type_length>([&](auto c) { result[c] *= scale[c]; });
    return result;
}
```

### 4.4. 错切变换  [[⬆](#目录)]

错切变换使物体沿某个轴倾斜，常用于创建斜投影效果。

## 5. 矩阵在图形管线中的作用  [[⬆](#目录)]

在3D图形管线中，矩阵按以下顺序应用：

1. **模型矩阵**: 将顶点从局部空间转换到世界空间
2. **视图矩阵**: 将顶点从世界空间转换到视图空间  
3. **投影矩阵**: 将顶点从视图空间转换到裁剪空间
4. **透视除法**: 将裁剪坐标转换为归一化设备坐标
5. **视口变换**: 将归一化坐标转换为屏幕坐标

**定义位置**: `source/blender/gpu/GPU_matrix.hh:145-147`

```cpp
const float (*GPU_matrix_model_view_projection_get(float m[4][4]))[4];
```

这个函数获取模型-视图-投影组合矩阵，是图形管线中的核心变换矩阵。

template<typename T, int NumCol, int NumRow, int Alignment = (((NumCol * NumRow) % 4 == 0) ? 4 : 1) * sizeof(T)>
struct alignas(Alignment) MatBase : public vec_struct_base<VecBase<T, NumRow>, NumCol, false> {
    using base_type = T;
    using vec3_type = VecBase<T, 3>;
    using col_type = VecBase<T, NumRow>;
    using row_type = VecBase<T, NumCol>;
    using loc_type = VecBase<T, (NumRow < NumCol) ? NumRow : (NumRow - 1)>;
    
    static constexpr int min_dim = (NumRow < NumCol) ? NumRow : NumCol;
    static constexpr int col_len = NumCol;
    static constexpr int row_len = NumRow;

    using float3x3 = MatBase<float, 3, 3>;

template<typename T, int NumCol, int NumRow>
[[nodiscard]] MatBase<T, NumCol, NumRow> transpose(const MatBase<T, NumRow, NumCol> &mat)
{
    MatBase<T, NumCol, NumRow> result;
    unroll<NumCol>([&](auto i) { unroll<NumRow>([&](auto j) { result[i][j] = mat[j][i]; }); });
    return result;
}

### 6.4. 投影矩阵实现  [[⬆](#目录)]

正交投影矩阵实现：

**定义位置**: `source/blender/blenlib/BLI_math_matrix.hh:1682-1699`

```cpp
template<typename T>
MatBase<T, 4, 4> orthographic(T left, T right, T bottom, T top, T near_clip, T far_clip)
{
    const T x_delta = right - left;
    const T y_delta = top - bottom;
    const T z_delta = far_clip - near_clip;

    MatBase<T, 4, 4> mat = MatBase<T, 4, 4>::identity();
    if (x_delta != 0 && y_delta != 0 && z_delta != 0) {
        mat[0][0] = T(2.0) / x_delta;
        mat[3][0] = -(right + left) / x_delta;
        mat[1][1] = T(2.0) / y_delta;
        mat[3][1] = -(top + bottom) / y_delta;
        mat[2][2] = -T(2.0) / z_delta; /* 注意：Z轴取反 */
        mat[3][2] = -(far_clip + near_clip) / z_delta;
    }
    return mat;
}
```

透视投影矩阵实现：

**定义位置**: `source/blender/blenlib/BLI_math_matrix.hh:1724-1743`

```cpp
template<typename T>
MatBase<T, 4, 4> perspective(T left, T right, T bottom, T top, T near_clip, T far_clip)
{
    const T x_delta = right - left;
    const T y_delta = top - bottom;
    const T z_delta = far_clip - near_clip;

    MatBase<T, 4, 4> mat = MatBase<T, 4, 4>::identity();
    if (x_delta != 0 && y_delta != 0 && z_delta != 0) {
        mat[0][0] = near_clip * T(2.0) / x_delta;
        mat[1][1] = near_clip * T(2.0) / y_delta;
        mat[2][0] = (right + left) / x_delta; /* 注意：Z轴取反 */
        mat[2][1] = (top + bottom) / y_delta;
        mat[2][2] = -(far_clip + near_clip) / z_delta;
        mat[2][3] = -1.0f;
        mat[3][2] = (-2.0f * near_clip * far_clip) / z_delta;
        mat[3][3] = 0.0f;
    }
    return mat;
}
```

## 7. 实际应用示例  [[⬆](#目录)]

### 7.1. 3D物体变换  [[⬆](#目录)]

创建一个包含位置、旋转和缩放的复合变换矩阵：

```cpp
// 创建变换矩阵
float3 location = {1.0f, 2.0f, 3.0f};
EulerXYZ rotation = {0.0f, M_PI/4, 0.0f}; // 绕Y轴旋转45度
float3 scale = {1.5f, 1.0f, 0.5f}; // 非均匀缩放

// 使用Blender的函数创建变换矩阵
float4x4 transform_matrix = math::from_loc_rot_scale<float4x4>(location, rotation, scale);

// 应用到顶点
float3 vertex = {0.0f, 0.0f, 0.0f};
float3 transformed_vertex = math::transform_point(transform_matrix, vertex);
```

### 7.2. 相机视图设置  [[⬆](#目录)]

设置相机视图矩阵：

```cpp
// 相机参数
float3 eye = {0.0f, 0.0f, 5.0f};    // 相机位置
float3 center = {0.0f, 0.0f, 0.0f};  // 观察目标
float3 up = {0.0f, 1.0f, 0.0f};      // 上方向

// 创建视图矩阵
float4x4 view_matrix = math::from_orthonormal_axes<float4x4>(
    normalize(center - eye),  // forward向量
    up                     // up向量
);
view_matrix.location() = eye;

// 或者使用GPU API
GPU_matrix_look_at(eye.x, eye.y, eye.z,
                 center.x, center.y, center.z,
                 up.x, up.y, up.z);
```

### 7.3. 投影设置  [[⬆](#目录)]

设置透视投影：

```cpp
// 透视投影参数
float fov = M_PI / 3.0f;  // 60度视野角
float aspect = 16.0f / 9.0f; // 宽高比
float near = 0.1f;
float far = 100.0f;

// 计算透视投影矩阵
float top = near * tan(fov / 2.0f);
float bottom = -top;
float right = top * aspect;
float left = -right;

float4x4 projection_matrix = math::projection::perspective(left, right, bottom, top, near, far);

// 或者使用GPU API
GPU_matrix_perspective_set(fov, aspect, near, far);
```

完整的MVP矩阵组合：

```cpp
// 组合模型、视图和投影矩阵
float4x4 model_matrix = math::from_loc_rot_scale<float4x4>(location, rotation, scale);
float4x4 view_matrix = create_view_matrix(eye, center, up);
float4x4 projection_matrix = math::projection::perspective(left, right, bottom, top, near, far);

// 计算最终的MVP矩阵
float4x4 mvp_matrix = projection_matrix * view_matrix * model_matrix;

// 应用到顶点
float4 clip_position = mvp_matrix * float4(vertex, 1.0f);
```

## 总结

矩阵是计算机图形学的核心数学工具，Blender提供了一套完整、高效的矩阵运算库。理解矩阵的类型、坐标空间和变换操作对于掌握3D图形编程至关重要。通过Blender的源码，我们可以学习到工业级的矩阵实现和优化技巧，这些知识可以应用到任何3D图形项目中。
