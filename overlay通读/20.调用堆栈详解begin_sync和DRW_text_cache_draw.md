## 概述

本文档详细分析Blender中两个关键函数的完整调用堆栈：`AttributeTexts::begin_sync` 和 `DRW_text_cache_draw`。这两个函数分别负责属性文本显示的同步初始化和文本缓存的实际绘制，是Blender几何节点属性查看器功能的核心组件。

### 12. 引擎初始化和同步：DRWContext::engines_init_and_sync()

**位置：** `source/blender/draw/intern/draw_context.cc:1002`

```cpp
void DRWContext::engines_init_and_sync(iter_callback_t iter_callback)
{
    /* 初始化所有启用的引擎 */
    view_data_active->foreach_enabled_engine([&](DrawEngine &instance) { 
        instance.init(); 
    });
    
    /* 开始同步 */
    view_data_active->manager->begin_sync(this->obact);
    
    /* 同步所有启用的引擎 */
    view_data_active->foreach_enabled_engine([&](DrawEngine &instance) { 
        instance.begin_sync(); 
    });
    
    /* 执行同步回调 */
    sync(iter_callback);
    
    /* 结束同步 */
    view_data_active->foreach_enabled_engine([&](DrawEngine &instance) { 
        instance.end_sync(); 
    });
    
    view_data_active->manager->end_sync();
}
```

**作用：**
- 管理所有绘制引擎的初始化和同步过程
- 按顺序执行init、begin_sync、sync、end_sync

**数据流：**
1. 初始化所有引擎
2. 开始管理器同步
3. 同步所有引擎
4. 执行自定义同步回调
5. 结束同步

```cpp
void Instance::draw_text(Framebuffer &framebuffer)
{
    if (state.show_text == false) {
        return;
    }
    GPU_framebuffer_bind(framebuffer);
    
    GPU_depth_test(GPU_DEPTH_NONE);
    DRW_text_cache_draw(state.dt, state.region, state.v3d);
}
```

**作用：**

- 绑定指定的帧缓冲区

```cpp
void DRW_text_cache_draw(const DRWTextStore *dt, const ARegion *region, const View3D *v3d)
{
    ViewCachedString *vos;
    if (v3d) {
        RegionView3D *rv3d = static_cast<RegionView3D *>(region->regiondata);
        int tot = 0;
        
        /* 首先投影并测试 */
        BLI_memiter_handle it;
        BLI_memiter_iter_init(dt->cache_strings, &it);
        while ((vos = static_cast<ViewCachedString *>(BLI_memiter_iter_step(&it)))) {
            if (ED_view3d_project_short_ex(
                    region,
                    (vos->flag & DRW_TEXT_CACHE_GLOBALSPACE) ? rv3d->persmat : rv3d->persmatob,
                    (vos->flag & DRW_TEXT_CACHE_LOCALCLIP) != 0,
                    vos->vec,
                    vos->sco,
                    V3D_PROJ_TEST_CLIP_BB | V3D_PROJ_TEST_CLIP_WIN | V3D_PROJ_TEST_CLIP_NEAR) ==
                V3D_PROJ_RET_OK)
            {
                tot++;
            }
            else {
                vos->sco[0] = IS_CLIPPED;
            }
        }
        
        if (tot) {
            /* 为文本禁用裁剪 */
            const bool rv3d_clipping_enabled = RV3D_CLIPPING_ENABLED(v3d, rv3d);
            if (rv3d_clipping_enabled) {
                GPU_clip_distances(0);
            }
            
            drw_text_cache_draw_ex(dt, region);
            
            if (rv3d_clipping_enabled) {
                GPU_clip_distances(6);
            }
        }
    }
}
```
