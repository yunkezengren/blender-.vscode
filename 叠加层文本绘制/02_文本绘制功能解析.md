## 概述

Blender的3D视图叠加层提供了丰富的文本绘制功能，用于在3D场景中显示各种信息，包括几何节点属性、测量数据、对象名称、索引编号等。这些文本绘制功能为用户提供了直观的场景信息反馈，是Blender用户界面中不可或缺的重要组成部分。

## 核心文本绘制系统

### 1. 基础架构  [[⬆](#目录)]

**文件位置**: `source/blender/draw/intern/draw_manager_text.cc`

**核心组件**:
```cpp
struct DRWTextStore {
  BLI_memiter *cache_strings;  // 文本缓存容器
};
```

**工作流程**:
1. **缓存阶段**: 各个overlay组件调用`DRW_text_cache_add()`添加文本
2. **投影阶段**: 将3D坐标投影到2D屏幕坐标
3. **渲染阶段**: 使用BLF库渲染所有缓存的文本

### 2. 文本绘制API  [[⬆](#目录)]

**核心函数**: `DRW_text_cache_add()`
```cpp
void DRW_text_cache_add(DRWTextStore *dt,
                        const float co[3],        // 3D世界坐标
                        const char *str,          // 文本字符串
                        int str_len,              // 字符串长度
                        short xoffs, yoffs,        // 像素偏移
                        short flag,               // 标志位
                        const uchar col[4],       // RGBA颜色
                        const bool shadow = false,     // 阴影效果
                        const bool align_center = false); // 居中对齐
```

**关键标志位**:
- `DRW_TEXT_CACHE_GLOBALSPACE`: 使用世界坐标系
- `DRW_TEXT_CACHE_LOCALCLIP`: 启用本地裁剪
- `DRW_TEXT_CACHE_STRING_PTR`: 使用字符串指针引用

### 3. 颜色系统和主题  [[⬆](#目录)]

**主要颜色主题**:

### 1. 边长测量  [[⬆](#目录)]

**功能**: 显示选中边的长度

```cpp
if (v3d->overlay.edit_flag & V3D_OVERLAY_EDIT_EDGE_LEN) {
  // 遍历选中的边
  BM_ITER_MESH (eed, &iter, em->bm, BM_EDGES_OF_MESH) {
    if (BM_elem_flag_test(eed, BM_ELEM_SELECT)) {
      // 计算边长度
      const float length = len_v3v3(v1, v2);
```

### 2. 角度测量  [[⬆](#目录)]

**功能**: 显示相邻面之间的角度
**启用条件**: `V3D_OVERLAY_EDIT_EDGE_ANG`
**颜色**: 主题颜色 `TH_DRAWEXTRA_EDGEANG`
**位置**: 边的中点位置

**算法实现**:
```cpp
// 获取边的两个相邻面
BMLoop *l_a, *l_b;
if (BM_edge_loop_pair(eed, &l_a, &l_b)) {
  // 计算两个面的法向量
  float3 no_a = l_a->f->no;
  float3 no_b = l_b->f->no;
  
  // 计算角度
  const float angle = angle_normalized_v3v3(no_a, no_b);
  
  // 格式化显示
  const size_t numstr_len = SNPRINTF_RLEN(numstr,
      "%.3f%s",
      (is_rad) ? angle : RAD2DEGF(angle),
      (is_rad) ? "r" : BLI_STR_UTF8_DEGREE_SIGN);
}
```

**显示示例**:

### 3. 面积测量  [[⬆](#目录)]

**功能**: 显示选中面的面积
**启用条件**: `V3D_OVERLAY_EDIT_FACE_AREA`
**颜色**: 主题颜色 `TH_DRAWEXTRA_FACEAREA`
**位置**: 面的中心位置

**计算方法**:
```cpp
// 通过三角化计算面积
float area = 0;
float3 vmid(0.0f);
const std::array<BMLoop *, 3> *ltri_array = &em->looptris[tri_index];

for (int j = 0; j < f_corner_tris_len; j++) {
  // 获取三角形的三个顶点
  float3 v1 = ltri_array[j][0]->v->co;
  float3 v2 = ltri_array[j][1]->v->co;
  float3 v3 = ltri_array[j][2]->v->co;
  
  // 累加面积和中心点
  area += area_tri_v3(v1, v2, v3);
  vmid += v1 + v2 + v3;
}
vmid /= float(n);  // 计算平均中心
```

**显示示例**:
```
2.456        // 默认单位
1.23m²       // 平方米
13.2ft²      // 平方英尺
```

```cpp
if (em->selectmode & SCE_SELECT_VERTEX) {
  BM_ITER_MESH_INDEX (v, &iter, em->bm, BM_VERTS_OF_MESH, i) {
```

### 2. 边索引  [[⬆](#目录)]

**偏移计算**:
```cpp
// 计算多个文本选项的间距
short edge_tex_count = 0;
if (v3d->overlay.edit_flag & V3D_OVERLAY_EDIT_EDGE_LEN) edge_tex_count += 1;
if (v3d->overlay.edit_flag & V3D_OVERLAY_EDIT_EDGE_ANG) edge_tex_count += 1;
if (v3d->overlay.edit_flag & V3D_OVERLAY_EDIT_INDICES) edge_tex_count += 1;

const short edge_tex_sep = short((edge_tex_count - 1) * 5.0f * UI_SCALE_FAC);
```

### 3. 性能优化  [[⬆](#目录)]

**批量渲染**:
```cpp
// 先批量投影所有文本
BLI_memiter_iter_init(dt->cache_strings, &it);
while ((vos = static_cast<ViewCachedString *>(BLI_memiter_iter_step(&it)))) {
  if (ED_view3d_project_short_ex(...) == V3D_PROJ_RET_OK) {
    tot++;  // 统计可见文本
  } else {
    vos->sco[0] = IS_CLIPPED;  // 标记为不可见
  }
}

// 只渲染可见的文本
if (tot) {
  drw_text_cache_draw_ex(dt, region);
}
```

**状态缓存**:

## 总结

Blender的3D视图叠加层文本绘制系统是一个功能强大、高度优化的信息显示系统。它具有以下特点：

1. **全面性**: 覆盖了从基础测量到高级调试的各种文本显示需求
