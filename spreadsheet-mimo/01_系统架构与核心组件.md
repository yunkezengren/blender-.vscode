# Blender 电子表格系统架构与核心组件

## 概述

Blender 的电子表格系统 (`space_spreadsheet`) 是一个用于显示和分析几何数据的编辑器。它提供了一个类似 Excel 的界面，用于查看几何体的属性、实例信息和其他数据。

## 核心数据结构

### 1. SpaceSpreadsheet (DNA 结构)

这是电子表格编辑器的主结构，存储在 `.blend` 文件中：

```cpp
/* 来自 DNA_space_types.h */
typedef struct SpaceSpreadsheet {
  SpaceLink *next, *prev;
  int spacetype;
  float regiondata;

  /* 视图控制 */
  float v2d.curzoom;
  float v2d.cur;

  /* 表格数据 */
  struct SpreadsheetTable **tables;
  int num_tables;

  /* 过滤器 */
  struct ListBase row_filters;
  short filter_flag;

  /* 活跃表格 ID */
  struct SpreadsheetTableID *active_table_id;

  /* 运行时数据 (不保存到文件) */
  struct SpaceSpreadsheet_Runtime *runtime;
} SpaceSpreadsheet;
```

**关键字段说明**:
- `tables`: 动态数组，存储所有表格
- `num_tables`: 表格数量
- `row_filters`: 链表，存储行过滤器
- `filter_flag`: 过滤器开关标志
- `active_table_id`: 当前显示的表格标识
- `runtime`: 运行时缓存数据

## 垃圾回收机制

### 表格垃圾回收

```cpp
/* 来自 spreadsheet_table.cc */
void spreadsheet_table_remove_unused(SpaceSpreadsheet &sspreadsheet) {
  uint32_t min_last_used = 0;
  const int max_tables = 50;

  /* 如果超过限制，计算 LRU 阈值 */
  if (sspreadsheet.num_tables > max_tables) {
    Vector<uint32_t> last_used_times;
    for (const SpreadsheetTable *table :
         Span(sspreadsheet.tables, sspreadsheet.num_tables)) {
      last_used_times.append(table->last_used);
    }
    std::sort(last_used_times.begin(), last_used_times.end());
    min_last_used = last_used_times[sspreadsheet.num_tables - max_tables];
  }

  /* 移除策略 */
  dna::array::remove_if<SpreadsheetTable *>(
      &sspreadsheet.tables,
      &sspreadsheet.num_tables,
      [&](const SpreadsheetTable *table) {
        /* 1. 移除从未手动编辑的表格 */
        if (!(table->flag & SPREADSHEET_TABLE_FLAG_MANUALLY_EDITED)) {
          return true;
        }

        /* 2. 移除长时间未使用的表格 */
        if (table->last_used < min_last_used) {
          return true;
        }

        /* 3. 移除引用已删除 ID 的表格 */
        switch (eSpreadsheetTableIDType(table->id->type)) {
          case SPREADSHEET_TABLE_ID_TYPE_GEOMETRY: {
            const SpreadsheetTableIDGeometry &table_id =
                *reinterpret_cast<const SpreadsheetTableIDGeometry *>(table->id);
            LISTBASE_FOREACH (ViewerPathElem *, elem, &table_id.viewer_path.path) {
              if (elem->type == VIEWER_PATH_ELEM_TYPE_ID) {
                const IDViewerPathElem &id_elem =
                    reinterpret_cast<const IDViewerPathElem &>(*elem);
                if (!id_elem.id) {
                  return true;
                }
              }
            }
            break;
          }
        }
        return false;
      },
      [](SpreadsheetTable **table) { spreadsheet_table_free(*table); });
}
```

### 列垃圾回收

```cpp
/* 来自 spreadsheet_table.cc */
void spreadsheet_table_remove_unused_columns(SpreadsheetTable &table) {
  const int max_unavailable_columns_target = 50;
  int num_unavailable_columns = 0;

  /* 统计不可用列数量 */
  for (SpreadsheetColumn *column : Span(table.columns, table.num_columns)) {
    if (!column->is_available()) {
      num_unavailable_columns++;
    }
  }

  if (num_unavailable_columns <= max_unavailable_columns_target) {
    return; /* 无需清理 */
  }

  /* 计算 LRU 阈值 */
  Vector<uint32_t> last_used_times;
  for (SpreadsheetColumn *column : Span(table.columns, table.num_columns)) {
    if (!column->is_available()) {
      last_used_times.append(column->last_used);
    }
  }
  std::sort(last_used_times.begin(), last_used_times.end());
  const int min_last_used = last_used_times[max_unavailable_columns_target];

  /* 移除旧的不可用列 */
  dna::array::remove_if<SpreadsheetColumn *>(
      &table.columns,
      &table.num_columns,
      [&](const SpreadsheetColumn *column) {
        if (column->is_available()) {
          return false; /* 可用列永不移除 */
        }
        if (column->last_used > min_last_used) {
          return false; /* 近期使用的保留 */
        }
        return true;
      },
      [](SpreadsheetColumn **column) { spreadsheet_column_free(*column); });
}
```

## 核心工作流程

### 1. 数据获取流程

```
用户打开电子表格编辑器
    ↓
space_spreadsheet.cc: spreadsheet_main_area_init()
    ↓
spreadsheet_intern.hh: get_data_source()
    ↓
spreadsheet_data_source_geometry.cc: 创建 GeometryDataSource
    ↓
根据 SpaceSpreadsheet 配置获取 GeometrySet
    ↓
根据 component_type, domain, layer_index 筛选数据
    ↓
返回 DataSource 实例
```

### 2. 列显示流程

```
DataSource::foreach_default_column_ids()
    ↓
为每个列创建 SpreadsheetColumn
    ↓
spreadsheet_column.cc: spreadsheet_column_assign_runtime_data()
    ↓
设置 display_name, data_type, width
    ↓
ColumnValues::fit_column_width_px() 计算宽度
    ↓
添加到表格的 columns 数组
```

### 3. 行过滤流程

```
spreadsheet_row_filter.cc: spreadsheet_filter_rows()
    ↓
获取所有 SpreadsheetRowFilter
    ↓
对每个过滤器调用 apply_row_filter()
    ↓
根据数据类型选择匹配函数
    ↓
使用 IndexMask::from_predicate() 并行过滤
    ↓
返回过滤后的行索引
```

### 4. 渲染流程

```
spreadsheet_draw.cc: spreadsheet_main_area_draw()
    ↓
计算可见区域 (scissor 测试)
    ↓
绘制表头 (header row)
    ↓
遍历可见行和列
    ↓
调用 draw_cell() 绘制每个单元格
    ↓
应用高亮和选择状态
```

## 内存管理策略

### 1. 所有权模型

- **DNA 结构**: `SpaceSpreadsheet` 和 `SpreadsheetRowFilter` 由 Blender 的 DNA 系统管理
- **动态数组**: `SpreadsheetTable**` 和 `SpreadsheetColumn**` 使用 `MEM_calloc_arrayN()`
- **字符串**: 使用 `BLI_strdup()` 和 `MEM_freeN()`
- **运行时对象**: 使用 `MEM_new()` 和 `MEM_delete()`

### 2. 复制语义

```cpp
/* 表格复制 */
SpreadsheetTable *spreadsheet_table_copy(const SpreadsheetTable &src_table) {
  SpreadsheetTable *new_table = spreadsheet_table_new(
      spreadsheet_table_id_copy(*src_table.id));
  new_table->num_columns = src_table.num_columns;
  new_table->columns = MEM_calloc_arrayN<SpreadsheetColumn *>(
      src_table.num_columns, __func__);
  for (const int i : IndexRange(src_table.num_columns)) {
    new_table->columns[i] = spreadsheet_column_copy(src_table.columns[i]);
  }
  return new_table;
}
```

### 3. 释放语义

```cpp
void spreadsheet_table_free(SpreadsheetTable *table) {
  spreadsheet_table_id_free(table->id);
  for (const int i : IndexRange(table->num_columns)) {
    spreadsheet_column_free(table->columns[i]);
  }
  MEM_SAFE_FREE(table->columns);
  MEM_freeN(table);
}
```

## 性能关键点

### 1. 延迟计算

- 列宽度在需要时才计算 (`fit_column_width_px`)
- 数据源在需要时才创建
- 表格缓存基于 LRU 策略

### 2. 索引掩码优化

```cpp
/* 使用 IndexMask 进行高效过滤 */
static IndexMask apply_filter_operation(const VArray<T> &data,
                                        OperationFn check_fn,
                                        const IndexMask &mask,
                                        IndexMaskMemory &memory) {
  return IndexMask::from_predicate(
      mask, GrainSize(1024), memory, [&](const int64_t i) { return check_fn(data[i]); });
}
```

### 3. 内存池管理

- 使用 `MEM_callocN()` 和 `MEM_freeN()` 管理所有内存
- 运行时结构使用 `MEM_new()` / `MEM_delete()`
- 数组使用 `MEM_calloc_arrayN()` / `MEM_SAFE_FREE()`

## 总结

Blender 电子表格系统的核心架构特点：

1. **分层设计**: DNA 结构 → 运行时结构 → 数据源抽象 → 具体实现
2. **类型安全**: 使用 C++ 模板和虚函数实现类型安全的多态
3. **内存安全**: 严格的内存所有权和释放策略
4. **性能优化**: LRU 缓存、延迟计算、索引掩码过滤
5. **可扩展**: 数据源模式支持多种几何类型和属性域
6. **持久化**: 完整的 Blend 文件序列化支持

**关键设计模式**:
- 工厂模式: `get_data_source()` 创建不同类型的数据源
- 策略模式: `DataSource` 接口的多种实现
- 观察者模式: ID 引用追踪和重映射
- LRU 缓存: 自动清理旧数据
