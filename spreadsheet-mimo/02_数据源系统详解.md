# Blender 电子表格数据源系统详解

## 概述

数据源系统是电子表格的核心，负责从几何数据中提取并组织信息以供显示。

## 数据源抽象层

### DataSource 基类定义

```cpp
/* 来源: spreadsheet_data_source.hh */
namespace blender::ed::spreadsheet {

class DataSource {
public:
  virtual ~DataSource() = default;

  /* 遍历默认列标识符 */
  virtual void foreach_default_column_ids(
      FunctionRef<void(const SpreadsheetColumnID &, bool is_extra)> fn) const = 0;

  /* 获取指定列的值 */
  virtual std::unique_ptr<ColumnValues> get_column_values(
      const SpreadsheetColumnID &column_id) const = 0;

  /* 获取总行数 */
  virtual int tot_rows() const = 0;

  /* 是否支持选择过滤 */
  virtual bool has_selection_filter() const;

  /* 应用选择过滤 */
  virtual IndexMask apply_selection_filter(IndexMaskMemory &memory) const;
};

}  // namespace blender::ed::spreadsheet
```

**关键方法说明**:

- `foreach_default_column_ids()`: 遍历所有可用的列，回调函数接收列ID和是否为额外列的标志
- `get_column_values()`: 获取特定列的数据，返回 `ColumnValues` 对象
- `tot_rows()`: 返回数据源的总行数
- `has_selection_filter()`: 检查是否支持基于选择的过滤
- `apply_selection_filter()`: 应用选择过滤，返回过滤后的索引掩码

## GeometryDataSource (几何数据源)

这是最常用的数据源，处理所有类型的几何数据。

### 类定义


## 数据源实现细节

### 1. 遍历默认列

```cpp
/* 来源: spreadsheet_data_source_geometry.cc */
void GeometryDataSource::foreach_default_column_ids(
    FunctionRef<void(const SpreadsheetColumnID &, bool is_extra)> fn) const
{
  /* 通用列: Index */
  fn({(char *)"Index"}, false);

  /* 根据组件类型添加特定列 */
  switch (component_type_) {
    case GEOMETRY_TYPE_MESH:
      add_mesh_columns(*geometry_set_->get_mesh(), domain_, fn);
      break;
    case GEOMETRY_TYPE_CURVES:
      add_curves_columns(*geometry_set_->get_curves(), domain_, fn);
      break;
    case GEOMETRY_TYPE_POINT_CLOUD:
      add_pointcloud_columns(*geometry_set_->get_pointcloud(), domain_, fn);
      break;
    case GEOMETRY_TYPE_INSTANCES:
      add_instances_columns(*geometry_set_->get_instances(), fn);
      break;
    /* ... 其他类型 */
  }

  /* Debug 列 */
  add_debug_columns(fn);
}
```

### 2. Mesh 数据源实现

```cpp
/* 来源: spreadsheet_data_source_geometry.cc */
static void add_mesh_columns(
    const Mesh &mesh,
    const bke::AttrDomain domain,
    FunctionRef<void(const SpreadsheetColumnID &, bool is_extra)> fn)
{
  switch (domain) {
    case bke::AttrDomain::Point:
      /* 顶点属性 */
      fn({(char *)"Position"}, false);
      fn({(char *)"Normal"}, false);
      if (CustomData_has_layer(&mesh.vert_data, CD_ORCO)) {
        fn({(char *)"Original Coordinates"}, false);
      }
      break;

    case bke::AttrDomain::Edge:
      /* 边属性 */
      fn({(char *)"Length"}, false);
      break;

    case bke::AttrDomain::Face:
      /* 面属性 */
      fn({(char *)"Area"}, false);
      fn({(char *)"Perimeter"}, false);
      break;

    case bke::AttrDomain::Corner:
      /* 角属性 */
      fn({(char *)"UV"}, false);
      fn({(char *)"Loop Normal"}, false);
      break;

    default:
      break;
  }
}
```

### 3. 获取列值

```cpp
/* 来源: spreadsheet_data_source_geometry.cc */
std::unique_ptr<ColumnValues> GeometryDataSource::get_column_values(
    const SpreadsheetColumnID &column_id) const
{
  const StringRef name = column_id.name;

  /* 通用 Index 列 */
  if (name == "Index") {
    return std::make_unique<ColumnValues>(
        "Index",
        VArray<int64_t>::from_range(0, tot_rows()));
  }

  /* 根据组件类型处理 */
  switch (component_type_) {
    case GEOMETRY_TYPE_MESH:
      return get_mesh_column_values(*geometry_set_->get_mesh(), domain_, name);
    case GEOMETRY_TYPE_CURVES:
      return get_curves_column_values(*geometry_set_->get_curves(), domain_, name);
    case GEOMETRY_TYPE_POINT_CLOUD:
      return get_pointcloud_column_values(*geometry_set_->get_pointcloud(), domain_, name);
    case GEOMETRY_TYPE_INSTANCES:
      return get_instances_column_values(*geometry_set_->get_instances(), name);
    /* ... 其他类型 */
  }

  return nullptr;
}
```

### 4. Mesh 列值获取示例

```cpp
/* 来源: spreadsheet_data_source_geometry.cc */
static std::unique_ptr<ColumnValues> get_mesh_column_values(
    const Mesh &mesh,
    const bke::AttrDomain domain,
    const StringRef name)
{
  if (domain == bke::AttrDomain::Point) {
    if (name == "Position") {
      const Span<float3> positions = mesh.positions();
      return std::make_unique<ColumnValues>(
          "Position",
          VArray<float3>::from_span(positions));
    }
    if (name == "Normal") {
      const Span<float3> normals = mesh.normals();
      return std::make_unique<ColumnValues>(
          "Normal",
          VArray<float3>::from_span(normals));
    }
  }
  else if (domain == bke::AttrDomain::Edge) {
    if (name == "Length") {
      const Span<float3> positions = mesh.positions();
      const Span<int2> edges = mesh.edges();
      return std::make_unique<ColumnValues>(
          "Length",
          VArray<float>::from_std_func(edges.size(), [&](int64_t i) {
            const int2 &edge = edges[i];
            return math::distance(positions[edge.x], positions[edge.y]);
          }));
    }
  }

  /* 从属性获取 */
  const bke::AttributeAccessor attributes = mesh.attributes();
  if (auto attr = attributes.lookup(name, domain)) {
    return std::make_unique<ColumnValues>(
        name,
        attr->varray);
  }

  return nullptr;
}
```

```cpp
static std::unique_ptr<ColumnValues> get_instances_column_values(
    const Instances &instances,
    const StringRef name)
{
  if (name == "Reference") {
    return std::make_unique<ColumnValues>(
        "Reference",

    return std::make_unique<ColumnValues>(
        "Transform",
        VArray<float4x4>::from_std_func(
            instances.instances_num(),
            [&](int64_t i) { return instances.get_instance_transform(i); }));
  }

  return nullptr;
}
```

```cpp
static void add_mesh_debug_column_names(

        fn({(char *)"Original Index"}, false);

      fn({(char *)"Corner Start"}, false);
      fn({(char *)"Corner Size"}, false);

  }
```

### 构建 Debug 列值

```cpp
/* 来源: spreadsheet_data_source_geometry.cc */
static std::unique_ptr<ColumnValues> build_mesh_debug_columns(
    const Mesh &mesh,
    const bke::AttrDomain domain,
    const StringRef name)
{
  if (domain == bke::AttrDomain::Point && name == "Original Index") {
    const int *data = static_cast<const int *>(
        CustomData_get_layer(&mesh.vert_data, CD_ORIGINDEX));
    if (data) {
      return std::make_unique<ColumnValues>(
          name,
          VArray<int>::from_span({data, mesh.verts_num}));
    }
  }

  if (domain == bke::AttrDomain::Face && name == "Corner Start") {
    return std::make_unique<ColumnValues>(
        name,
        VArray<int>::from_span(mesh.face_offsets().drop_back(1)));
  }

  if (domain == bke::AttrDomain::Face && name == "Corner Size") {
    const OffsetIndices faces = mesh.faces();
    return std::make_unique<ColumnValues>(
        name,
        VArray<int>::from_std_func(faces.size(), [faces](int64_t index) {
          return faces[index].size();
        }));
  }

  return nullptr;
}
```

## 选择过滤实现

### 选择过滤检查

```cpp
/* 来源: spreadsheet_data_source_geometry.cc */
bool GeometryDataSource::has_selection_filter() const
{
  /* 只有在编辑模式下且有选中项时才支持 */
  if (component_type_ == GEOMETRY_TYPE_MESH) {
    const Mesh *mesh = geometry_set_->get_mesh();
    if (mesh && mesh->runtime->edit_mesh) {
      return true;
    }
  }
  return false;
}
```

### 应用选择过滤

```cpp
/* 来源: spreadsheet_data_source_geometry.cc */
IndexMask GeometryDataSource::apply_selection_filter(IndexMaskMemory &memory) const
{
  if (!has_selection_filter()) {
    return IndexMask(tot_rows());
  }

  switch (component_type_) {
    case GEOMETRY_TYPE_MESH: {
      const Mesh *mesh = geometry_set_->get_mesh();
      const BMesh *bm = mesh->runtime->edit_mesh->bm;

      switch (domain_) {
        case bke::AttrDomain::Point: {
          Vector<int64_t> selected_indices;
          BMVert *vert;
          BM_ITER_MESH (vert, nullptr, bm, BM_VERTS_OF_MESH) {
            if (!BM_elem_flag_test(vert, BM_ELEM_SELECT)) {
              continue;
            }
            selected_indices.append(BM_elem_index_get(vert));
          }
          return IndexMask::from_vector(selected_indices, memory);
        }
        case bke::AttrDomain::Edge: {
          Vector<int64_t> selected_indices;
          BMEdge *edge;
          BM_ITER_MESH (edge, nullptr, bm, BM_EDGES_OF_MESH) {
            if (!BM_elem_flag_test(edge, BM_ELEM_SELECT)) {
              continue;
            }
            selected_indices.append(BM_elem_index_get(edge));
          }
          return IndexMask::from_vector(selected_indices, memory);
        }
        case bke::AttrDomain::Face: {
          Vector<int64_t> selected_indices;
          BMFace *face;
          BM_ITER_MESH (face, nullptr, bm, BM_FACES_OF_MESH) {
            if (!BM_elem_flag_test(face, BM_ELEM_SELECT)) {
              continue;
            }
            selected_indices.append(BM_elem_index_get(face));
          }
          return IndexMask::from_vector(selected_indices, memory);
        }
        default:
          break;
      }
      break;
    }
    /* 其他组件类型 */
  }

  return IndexMask(tot_rows());
}
```

```cpp
struct SpreadsheetBundlePathElem {
  char *identifier;  /* Bundle 字段名 */

bool operator==(const SpreadsheetBundlePathElem &a, const SpreadsheetBundlePathElem &b)
{
  return STREQ(a.identifier, b.identifier);
```

### Bundle 数据获取

```cpp
/* 来源: spreadsheet_data_source_geometry.cc */
static std::unique_ptr<ColumnValues> get_bundle_column_values(
    const nodes::Bundle &bundle,
    const Span<SpreadsheetBundlePathElem> path)
{
  /* 导航到 Bundle 的指定字段 */
  const nodes::Bundle *current = &bundle;
  for (const auto &elem : path) {
    current = current->get_field(elem.identifier);
    if (!current) {
      return nullptr;
    }
  }

  /* 获取字段值 */
  const CPPType &type = current->get_type();
  return std::make_unique<ColumnValues>(
      current->get_name(),
      current->get_varray());
}
```

## 数据源创建流程

### 从上下文获取数据源

```cpp
/* 来源: spreadsheet_intern.hh */
std::unique_ptr<DataSource> get_data_source(const bContext &C);

/* 来源: spreadsheet_data_source.cc */
std::unique_ptr<DataSource> get_data_source(const bContext &C)
{
  SpaceSpreadsheet *sspreadsheet = CTX_wm_space_spreadsheet(&C);
  if (!sspreadsheet) {
    return nullptr;
  }

  /* 获取活跃表格 ID */
  const SpreadsheetTableID *table_id = get_active_table_id(*sspreadsheet);
  if (!table_id) {
    return nullptr;
  }

  /* 根据类型创建数据源 */
  switch (eSpreadsheetTableIDType(table_id->type)) {
    case SPREADSHEET_TABLE_ID_TYPE_GEOMETRY: {
      const auto &geo_id = *reinterpret_cast<const SpreadsheetTableIDGeometry *>(table_id);

      /* 获取几何集 */
      const GeometrySet *geometry_set = geometry_set_get_from_viewer_path(
          &geo_id.viewer_path);
      if (!geometry_set) {
        return nullptr;
      }

      /* 创建几何数据源 */
      return std::make_unique<GeometryDataSource>(
          geometry_set,
          geo_id.geometry_component_type,
          geo_id.attribute_domain,
          geo_id.layer_index);
    }
  }

  return nullptr;
}
```

### 从 ViewerPath 获取 GeometrySet

```cpp
/* 来源: spreadsheet_data_source.cc */
static const GeometrySet *geometry_set_get_from_viewer_path(
    const ViewerPath *viewer_path)
{
  /* 遍历 ViewerPath 查找几何数据 */
  LISTBASE_FOREACH (ViewerPathElem *, elem, &viewer_path->path) {
    if (elem->type == VIEWER_PATH_ELEM_TYPE_ID) {
      const IDViewerPathElem &id_elem = reinterpret_cast<const IDViewerPathElem &>(*elem);
      if (id_elem.id) {
        /* 从 ID 获取几何集 */
        return geometry_set_from_id(id_elem.id);
      }
    }
    else if (elem->type == VIEWER_PATH_ELEM_TYPE_GEOMETRY_SET) {
      const GeometrySetViewerPathElem &geo_elem =
          reinterpret_cast<const GeometrySetViewerPathElem &>(*elem);
      return &geo_elem.geometry_set;
    }
  }
  return nullptr;
}
```

## 性能考虑

### 1. 延迟计算

```cpp
/* ColumnValues 只在需要时创建 */
std::unique_ptr<ColumnValues> values = data_source->get_column_values(column_id);
```

### 2. 数据类型转换

```cpp
/* 使用 GVArray 进行类型擦除 */
GVArray data = column.data();
const CPPType &type = data.type();

/* 只在需要时转换 */
if (type.is<float3>()) {
  const VArray<float3> &typed = data.typed<float3>();
  /* 使用 typed 数据 */
}
```

### 3. 索引掩码优化

```cpp
/* 使用 IndexMask 进行高效过滤 */
IndexMask mask = data_source->apply_selection_filter(mask_memory);
mask = apply_row_filter(*row_filter, columns, mask, mask_memory);
```

## 总结

数据源系统的关键特点：

1. **多态设计**: `DataSource` 接口支持多种数据源类型
2. **类型安全**: 使用 C++ 模板和虚函数
3. **高效**: 延迟计算、索引掩码、类型擦除
4. **可扩展**: 易于添加新的几何类型和属性域
5. **集成**: 与 Blender 的 GeometrySet、属性系统深度集成

**核心流程**:
```
ViewerPath → GeometrySet → GeometryDataSource → ColumnValues → UI
```
