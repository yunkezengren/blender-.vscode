# Blender 电子表格内存管理与序列化

## 概述

内存管理和序列化系统确保电子表格数据的正确分配、复制、释放以及在 Blend 文件中的持久化存储。

## 复制语义

### 深拷贝 vs 浅拷贝

#### 深拷贝 (Deep Copy)

```cpp
/* 来源: spreadsheet_column.cc */
SpreadsheetColumn *spreadsheet_column_copy(const SpreadsheetColumn *src_column)
{
  /* 1. 复制列 ID (深拷贝) */
  SpreadsheetColumnID *new_column_id = spreadsheet_column_id_copy(src_column->id);

  /* 2. 创建新列 */
  SpreadsheetColumn *new_column = spreadsheet_column_new(new_column_id);

  /* 3. 复制显示名称 (深拷贝) */
  if (src_column->display_name != nullptr) {
    new_column->display_name = BLI_strdup(src_column->display_name);
  }

  /* 4. 复制基本类型 */
  new_column->width = src_column->width;
  new_column->data_type = src_column->data_type;

  return new_column;
}
```

#### 浅拷贝 (Shallow Copy)

```cpp
/* DNA 结构复制 */
SpaceSpreadsheet *sspreadsheet_new = (SpaceSpreadsheet *)MEM_dupallocN(sspreadsheet_old);

/* 注意: 这会复制指针，但不会复制指针指向的数据 */
```

### 复制层次结构

```
SpreadsheetTable (源)
  ├─ SpreadsheetTableID* (深拷贝)
  │   └─ SpreadsheetTableIDGeometry
  │       ├─ ViewerPath (深拷贝)
  │       ├─ SpreadsheetInstanceID* (深拷贝数组)
  │       └─ SpreadsheetBundlePathElem* (深拷贝数组 + 字符串)
  └─ SpreadsheetColumn** (数组，深拷贝)
      └─ SpreadsheetColumn
          ├─ SpreadsheetColumnID* (深拷贝)
          └─ char* display_name (深拷贝)
```

## 释放策略

### 递归释放

```cpp
/* 来源: spreadsheet_column.cc */
void spreadsheet_column_free(SpreadsheetColumn *column)
{
  spreadsheet_column_id_free(column->id);  /* 释放 ID */
  MEM_SAFE_FREE(column->display_name);    /* 释放字符串 */
  MEM_delete(column->runtime);            /* 释放运行时 */
  MEM_freeN(column);                      /* 释放结构 */
}

/* 来源: spreadsheet_table.cc */
void spreadsheet_table_free(SpreadsheetTable *table)
{
  spreadsheet_table_id_free(table->id);  /* 释放 ID (递归) */

  for (const int i : IndexRange(table->num_columns)) {
    spreadsheet_column_free(table->columns[i]);  /* 释放每列 */
  }

  MEM_SAFE_FREE(table->columns);  /* 释放列数组 */
  MEM_freeN(table);               /* 释放表格 */
}
```

### SAFE_FREE 宏

```cpp
/* 来源: MEM_guardedalloc.h */
#define MEM_SAFE_FREE(ptr) { \
  if (ptr) { \
    MEM_freeN(ptr); \
    (ptr) = nullptr; \
  } \
}
```

### 在电子表格中的应用

```cpp
/* 来源: spreadsheet_table.cc */
void spreadsheet_table_id_free_content(SpreadsheetTableID *table_id)
{
  switch (eSpreadsheetTableIDType(table_id->type)) {
    case SPREADSHEET_TABLE_ID_TYPE_GEOMETRY: {
      auto *table_id_ = reinterpret_cast<SpreadsheetTableIDGeometry *>(table_id);

      BKE_viewer_path_clear(&table_id_->viewer_path);
      MEM_SAFE_FREE(table_id_->instance_ids);  /* 释放实例 ID 数组 */

      /* 释放 Bundle 路径数组 */
      for (const int i : IndexRange(table_id_->bundle_path_num)) {
        MEM_SAFE_FREE(table_id_->bundle_path[i].identifier);
      }
      MEM_SAFE_FREE(table_id_->bundle_path);

      break;
    }
  }
}
```

## 序列化实现

### 表格序列化

#### 写入

```cpp
/* 来源: spreadsheet_table.cc */
void spreadsheet_table_blend_write(BlendWriter *writer, const SpreadsheetTable *table)
{
  /* 1. 写入表格结构 */
  BLO_write_struct(writer, SpreadsheetTable, table);

  /* 2. 写入表格 ID */
  spreadsheet_table_id_blend_write(writer, table->id);

  /* 3. 写入列指针数组 */
  BLO_write_pointer_array(writer, table->num_columns, table->columns);

  /* 4. 写入每列 */
  for (const int i : IndexRange(table->num_columns)) {
    spreadsheet_column_blend_write(writer, table->columns[i]);
  }
}
```

#### 读取

```cpp
/* 来源: spreadsheet_table.cc */
void spreadsheet_table_blend_read(BlendDataReader *reader, SpreadsheetTable *table)
{
  /* 1. 读取表格 ID */
  BLO_read_struct(reader, SpreadsheetTableID, &table->id);
  spreadsheet_table_id_blend_read(reader, table->id);

  /* 2. 读取列指针数组 */
  BLO_read_pointer_array(reader, table->num_columns,
                        reinterpret_cast<void **>(&table->columns));

  /* 3. 读取每列 */
  for (const int i : IndexRange(table->num_columns)) {
    BLO_read_struct(reader, SpreadsheetColumn, &table->columns[i]);
    spreadsheet_column_blend_read(reader, table->columns[i]);
  }
}
```

### 表格 ID 序列化

#### 几何表格 ID 写入

```cpp
/* 来源: spreadsheet_table.cc */
void spreadsheet_table_id_blend_write(BlendWriter *writer, const SpreadsheetTableID *table_id)
{
  switch (eSpreadsheetTableIDType(table_id->type)) {
    case SPREADSHEET_TABLE_ID_TYPE_GEOMETRY: {
      const auto *table_id_ = reinterpret_cast<const SpreadsheetTableIDGeometry *>(table_id);

      /* 1. 写入结构 */
      BLO_write_struct(writer, SpreadsheetTableIDGeometry, table_id_);

      /* 2. 写入 ViewerPath */
      BKE_viewer_path_blend_write(writer, &table_id_->viewer_path);

      /* 3. 写入实例 ID 数组 */
      BLO_write_struct_array(
          writer, SpreadsheetInstanceID, table_id_->instance_ids_num, table_id_->instance_ids);

      /* 4. 写入 Bundle 路径数组 */
      BLO_write_struct_array(
          writer, SpreadsheetBundlePathElem, table_id_->bundle_path_num, table_id_->bundle_path);

      /* 5. 写入 Bundle 标识符字符串 */
      for (const int i : IndexRange(table_id_->bundle_path_num)) {
        BLO_write_string(writer, table_id_->bundle_path[i].identifier);
      }
      break;
    }
  }
}
```

#### 几何表格 ID 读取

```cpp
/* 来源: spreadsheet_table.cc */
void spreadsheet_table_id_blend_read(BlendDataReader *reader, SpreadsheetTableID *table_id)

      /* 1. 读取 ViewerPath */
      BKE_viewer_path_blend_read_data(reader, &table_id_->viewer_path);
```

### SpaceSpreadsheet 序列化

#### 写入

```cpp
/* 来源: space_spreadsheet.cc */
static void spreadsheet_blend_write(BlendWriter *writer, SpaceLink *sl)
{
  BLO_write_struct(writer, SpaceSpreadsheet, sl);
  SpaceSpreadsheet *sspreadsheet = (SpaceSpreadsheet *)sl;

  /* 1. 写入行过滤器链表 */
  LISTBASE_FOREACH (SpreadsheetRowFilter *, row_filter, &sspreadsheet->row_filters) {
    BLO_write_struct(writer, SpreadsheetRowFilter, row_filter);
    BLO_write_string(writer, row_filter->value_string);
  }

  /* 2. 写入表格数组 */
  BLO_write_pointer_array(writer, sspreadsheet->num_tables, sspreadsheet->tables);
  for (const int i : IndexRange(sspreadsheet->num_tables)) {
    spreadsheet_table_blend_write(writer, sspreadsheet->tables[i]);
  }

  /* 3. 写入活跃表格 ID 内容 */
  spreadsheet_table_id_blend_write_content_geometry(writer, &sspreadsheet->geometry_id);
}
```

#### 读取

```cpp
/* 来源: space_spreadsheet.cc */
static void spreadsheet_blend_read_data(BlendDataReader *reader, SpaceLink *sl)
{
  SpaceSpreadsheet *sspreadsheet = (SpaceSpreadsheet *)sl;

  /* 1. 重建运行时 */
  sspreadsheet->runtime = MEM_new<SpaceSpreadsheet_Runtime>(__func__);

  /* 2. 读取行过滤器链表 */
  BLO_read_struct_list(reader, SpreadsheetRowFilter, &sspreadsheet->row_filters);
  LISTBASE_FOREACH (SpreadsheetRowFilter *, row_filter, &sspreadsheet->row_filters) {
    BLO_read_string(reader, &row_filter->value_string);
  }

  /* 3. 读取表格数组 */
  BLO_read_pointer_array(
      reader, sspreadsheet->num_tables, reinterpret_cast<void **>(&sspreadsheet->tables));
  for (const int i : IndexRange(sspreadsheet->num_tables)) {
    BLO_read_struct(reader, SpreadsheetTable, &sspreadsheet->tables[i]);
    spreadsheet_table_blend_read(reader, sspreadsheet->tables[i]);
  }

  /* 4. 读取活跃表格 ID */
  spreadsheet_table_id_blend_read(reader, &sspreadsheet->geometry_id.base);
}
```

### SpaceSpreadsheet 的 ID 管理

```cpp
/* 来源: space_spreadsheet.cc */
static void spreadsheet_id_remap(ScrArea * /*area*/,
                                 SpaceLink *slink,
                                 const blender::bke::id::IDRemapper &mappings)
{
  SpaceSpreadsheet *sspreadsheet = (SpaceSpreadsheet *)slink;

  /* 重映射活跃表格 ID */
  spreadsheet_table_id_remap_id(sspreadsheet->geometry_id.base, mappings);

    spreadsheet_table_remap_id(*sspreadsheet->tables[i], mappings);
  }
}

  /* 遍历活跃表格 ID */
  spreadsheet_table_id_foreach_id(sspreadsheet->geometry_id.base, data);

  /* 遍历所有表格 */
  for (const int i : IndexRange(sspreadsheet->num_tables)) {
    spreadsheet_table_foreach_id(*sspreadsheet->tables[i], data);
  }
}
```

## 完整的生命周期

### 1. 创建

```cpp
/* 创建空间 */
SpaceSpreadsheet *sspreadsheet = MEM_callocN<SpaceSpreadsheet>("spreadsheet space");
sspreadsheet->runtime = MEM_new<SpaceSpreadsheet_Runtime>(__func__);

/* 创建表格 ID */
sspreadsheet->geometry_id.base.type = SPREADSHEET_TABLE_ID_TYPE_GEOMETRY;

/* 创建表格 */
SpreadsheetTable *table = spreadsheet_table_new(
    spreadsheet_table_id_copy(sspreadsheet->geometry_id.base));

/* 添加到数组 */
spreadsheet_table_add(*sspreadsheet, table);
```

### 2. 复制 (Duplicate)

```cpp
/* 空间复制 */
SpaceSpreadsheet *sspreadsheet_new = (SpaceSpreadsheet *)MEM_dupallocN(sspreadsheet_old);
sspreadsheet_new->runtime = MEM_new<SpaceSpreadsheet_Runtime>(__func__,
                                                              *sspreadsheet_old->runtime);

/* 复制过滤器链表 */
LISTBASE_FOREACH (const SpreadsheetRowFilter *, src_filter, &sspreadsheet_old->row_filters) {
  SpreadsheetRowFilter *new_filter = spreadsheet_row_filter_copy(src_filter);
  BLI_addtail(&sspreadsheet_new->row_filters, new_filter);
}

/* 复制表格数组 */
for (const int i : IndexRange(sspreadsheet_old->num_tables)) {
  sspreadsheet_new->tables[i] = spreadsheet_table_copy(*sspreadsheet_old->tables[i]);
}

/* 复制表格 ID 内容 */
spreadsheet_table_id_copy_content_geometry(sspreadsheet_new->geometry_id,
                                           sspreadsheet_old->geometry_id);
```

### 3. 释放 (Free)

```cpp
/* 来源: space_spreadsheet.cc */
static void spreadsheet_free(SpaceLink *sl)
{
  SpaceSpreadsheet *sspreadsheet = (SpaceSpreadsheet *)sl;

  /* 1. 释放运行时 */
  MEM_delete(sspreadsheet->runtime);

  /* 2. 释放过滤器链表 */
  LISTBASE_FOREACH_MUTABLE (SpreadsheetRowFilter *, row_filter, &sspreadsheet->row_filters) {
    spreadsheet_row_filter_free(row_filter);
  }

  /* 3. 释放表格数组 */
  for (const int i : IndexRange(sspreadsheet->num_tables)) {
    spreadsheet_table_free(sspreadsheet->tables[i]);
  }
  MEM_SAFE_FREE(sspreadsheet->tables);

  /* 4. 释放表格 ID 内容 */
  spreadsheet_table_id_free_content(&sspreadsheet->geometry_id.base);
}
```

### 4. 序列化 (Read/Write)

```cpp
/* 写入 */
spreadsheet_blend_write(writer, sl);

/* 读取 */
```

## 内存泄漏检测

### Blender 内存调试

```cpp
/* 启用内存调试 (编译时) */
/* 在 CMake 中: -DWITH_MEM_DEBUG=ON */

/* 运行时检查 */
MEM_printmemlist();  /* 打印内存使用情况 */
MEM_check();         /* 检查内存完整性 */
```

### 常见泄漏点

1. **忘记释放字符串**:
   ```cpp
   /* 错误 */
   char *str = BLI_strdup("test");
   /* 忘记 MEM_freeN(str) */

   /* 正确 */
   char *str = BLI_strdup("test");
   MEM_freeN(str);
   ```

2. **忘记释放运行时数据**:
   ```cpp
   /* 错误 */
   column->runtime = MEM_new<SpreadsheetColumnRuntime>(__func__);
   /* 忘记 MEM_delete(column->runtime) */

   /* 正确 */
   column->runtime = MEM_new<SpreadsheetColumnRuntime>(__func__);
   /* ... */
   MEM_delete(column->runtime);
   ```

3. **数组释放不完整**:
   ```cpp
   /* 错误 */
   MEM_freeN(table->columns);  /* 忘记释放每个元素 */

   /* 正确 */
   for (const int i : IndexRange(table->num_columns)) {
     spreadsheet_column_free(table->columns[i]);
   }
   MEM_freeN(table->columns);
   ```

## 性能考虑

### 1. 内存分配优化

```cpp
/* 批量分配数组 */
table->columns = MEM_calloc_arrayN<SpreadsheetColumn *>(num_columns, __func__);

/* 而非循环分配 */
for (int i = 0; i < num_columns; i++) {
  table->columns[i] = MEM_callocN<SpreadsheetColumn>(__func__);  /* 低效 */
}
```

### 2. 避免不必要的复制

```cpp
/* 只在需要时复制 */
if (column->display_name != nullptr) {
  new_column->display_name = BLI_strdup(src_column->display_name);
}
```

### 3. 使用 Span 避免拷贝

```cpp
/* 使用 Span 访问数组，不复制 */
Span<SpreadsheetColumn *> columns(table.columns, table.num_columns);
for (SpreadsheetColumn *column : columns) {
  /* 处理列 */
}
```

## 错误处理

### 1. 空指针检查

```cpp
void spreadsheet_table_free(SpreadsheetTable *table)
{
  if (table == nullptr) {
    return;
  }

  spreadsheet_table_id_free(table->id);

  if (table->columns) {
    for (const int i : IndexRange(table->num_columns)) {
      if (table->columns[i]) {
        spreadsheet_column_free(table->columns[i]);
      }
    }
    MEM_freeN(table->columns);
  }

  MEM_freeN(table);
}
```

```cpp
SpreadsheetColumn **columns = MEM_calloc_arrayN<SpreadsheetColumn *>(num, __func__);
if (columns == nullptr) {
```

### 3. 序列化完整性

```cpp
void spreadsheet_column_blend_read(BlendDataReader *reader, SpreadsheetColumn *column)

  if (column == nullptr) {
```

## 总结

内存管理与序列化系统的关键特点：

1. **严格所有权**: 清晰的内存所有权模型
2. **深拷贝**: 复杂结构的完整复制
3. **自动释放**: 递归释放所有关联内存
4. **持久化**: 完整的 Blend 文件支持
5. **引用追踪**: ID 重映射和遍历
6. **错误安全**: 空指针和失败处理

**核心原则**:
- 每个 `MEM_callocN()` 对应一个 `MEM_freeN()`
- 每个 `BLI_strdup()` 对应一个 `MEM_freeN()`
- 每个 `MEM_new()` 对应一个 `MEM_delete()`
- 复制时必须深拷贝所有指针
- 释放时必须递归释放所有子结构

**序列化规则**:
- DNA 结构自动序列化
- 指针需要手动读写
- 字符串需要手动读写
- 运行时数据不序列化
- 链表需要特殊处理
