# Blender 电子表格列与值类型系统

## 概述

列与值类型系统负责管理电子表格中显示的数据类型、列标识符和值格式化。

## 类型转换系统

### C++ 类型到列类型

```cpp
/* 来源: spreadsheet_column.cc */
eSpreadsheetColumnValueType cpp_type_to_column_type(const CPPType &type)
{
  if (type.is<bool>()) {
    return SPREADSHEET_VALUE_TYPE_BOOL;
  }
  if (type.is<int8_t>()) {
    return SPREADSHEET_VALUE_TYPE_INT8;
  }
  if (type.is<int>()) {
    return SPREADSHEET_VALUE_TYPE_INT32;
  }
  if (type.is<int64_t>()) {
    return SPREADSHEET_VALUE_TYPE_INT64;
  }
  if (type.is_any<short2, int2>()) {
    return SPREADSHEET_VALUE_TYPE_INT32_2D;
  }
  if (type.is<int3>()) {
    return SPREADSHEET_VALUE_TYPE_INT32_3D;
  }
  if (type.is<float>()) {
    return SPREADSHEET_VALUE_TYPE_FLOAT;
  }
  if (type.is<float2>()) {
    return SPREADSHEET_VALUE_TYPE_FLOAT2;
  }
  if (type.is<float3>()) {
    return SPREADSHEET_VALUE_TYPE_FLOAT3;
  }
  if (type.is<ColorGeometry4f>()) {
    return SPREADSHEET_VALUE_TYPE_COLOR;
  }
  if (type.is<std::string>() || type.is<MStringProperty>()) {
    return SPREADSHEET_VALUE_TYPE_STRING;
  }
  if (type.is<bke::InstanceReference>()) {
    return SPREADSHEET_VALUE_TYPE_INSTANCES;
  }
  if (type.is<ColorGeometry4b>()) {
    return SPREADSHEET_VALUE_TYPE_BYTE_COLOR;
  }
  if (type.is<math::Quaternion>()) {
    return SPREADSHEET_VALUE_TYPE_QUATERNION;
  }
  if (type.is<float4x4>()) {
    return SPREADSHEET_VALUE_TYPE_FLOAT4X4;
  }
  if (type.is<nodes::BundleItemValue>()) {
    return SPREADSHEET_VALUE_TYPE_BUNDLE_ITEM;
  }

  return SPREADSHEET_VALUE_TYPE_UNKNOWN;
}
```

### 定义

```cpp
/* 来源: spreadsheet_column_values.hh */
enum class ColumnValueDisplayHint {
  None,
  Bytes,
};

class ColumnValues final {
protected:
  std::string name_;
  GVArray data_;
  ColumnValueDisplayHint display_hint_;

public:
  ColumnValues(std::string name,
               GVArray data,
               ColumnValueDisplayHint display_hint = ColumnValueDisplayHint::None)
      : name_(std::move(name)), data_(std::move(data)), display_hint_(display_hint)
  {
    BLI_assert(data_);  /* 数组不应为空 */
  }

  virtual ~ColumnValues() = default;

  /* 访问器 */
  eSpreadsheetColumnValueType type() const
  {
    return cpp_type_to_column_type(data_.type());
  }

  StringRefNull name() const
  {
    return name_;
  }

  int size() const
  {
    return data_.size();
  }

  const GVArray &data() const
  {
    return data_;
  }

  ColumnValueDisplayHint display_hint() const
  {
    return display_hint_;
  }

  /* 列宽计算 */
  float fit_column_width_px(const std::optional<int64_t> &max_sample_size = std::nullopt) const;
  float fit_column_values_width_px(const std::optional<int64_t> &max_sample_size = std::nullopt) const;
};
```

## 列宽计算

### 实现原理

```cpp
/* 来源: spreadsheet_column_values.cc (概念实现) */
float ColumnValues::fit_column_width_px(const std::optional<int64_t> &max_sample_size) const
{
  /* 1. 计算列名宽度 */
  float name_width = calculate_text_width(name_);

  /* 2. 计算值宽度 (采样) */
  float values_width = fit_column_values_width_px(max_sample_size);

  /* 3. 返回最大值 */
  return std::max(name_width, values_width);
}

float ColumnValues::fit_column_values_width_px(
    const std::optional<int64_t> &max_sample_size) const
{
  /* 确定采样大小 */
  int64_t sample_size = size();
  if (max_sample_size && sample_size > *max_sample_size) {
    sample_size = *max_sample_size;
  }

  /* 遍历采样值计算最大宽度 */
  float max_width = 0.0f;
  for (int64_t i = 0; i < sample_size; ++i) {
    std::string formatted = format_value(i);
    float width = calculate_text_width(formatted);
    max_width = std::max(max_width, width);
  }

  /* 最小宽度限制 */
  return std::max(max_width, MIN_COLUMN_WIDTH);
}
```

### 值格式化

```cpp
/* 来源: spreadsheet_draw.cc */
static std::string format_value(const GPointer &value, const CPPType &type)
{
  if (type.is<float>()) {
    float v = *static_cast<const float *>(value);
    return fmt::format("{:.6f}", v);
  }
  else if (type.is<float3>()) {
    const float3 &v = *static_cast<const float3 *>(value);
    return fmt::format("({:.3f}, {:.3f}, {:.3f})", v.x, v.y, v.z);
  }
  else if (type.is<int>()) {
    int v = *static_cast<const int *>(value);
    return std::to_string(v);
  }
  else if (type.is<bool>()) {
    bool v = *static_cast<const bool *>(value);
    return v ? "True" : "False";
  }
  else if (type.is<ColorGeometry4f>()) {
    const ColorGeometry4f &v = *static_cast<const ColorGeometry4f *>(value);
    return fmt::format("({:.3f}, {:.3f}, {:.3f}, {:.3f})", v.r, v.g, v.b, v.a);
  }
  else if (type.is<std::string>()) {
    const std::string &v = *static_cast<const std::string *>(value);
    return v;
  }
  else if (type.is<float4x4>()) {
    const float4x4 &v = *static_cast<const float4x4 *>(value);
    return format_matrix_to_grid(v);  /* 矩阵格式化 */
  }
  else if (type.is<bke::InstanceReference>()) {
    const bke::InstanceReference &v = *static_cast<const bke::InstanceReference *>(value);
    switch (v.type()) {
      case bke::InstanceReference::Type::Object:
        return reinterpret_cast<ID &>(v.object()).name + 2;
      case bke::InstanceReference::Type::Collection:
        return reinterpret_cast<ID &>(v.collection()).name + 2;
      case bke::InstanceReference::Type::GeometrySet:
        return v.geometry_set().name;
      case bke::InstanceReference::Type::None:
        return "None";
    }
  }

  return "Unknown";
}
```

```cpp
SpreadsheetColumn *spreadsheet_column_copy(const SpreadsheetColumn *src_column)

  SpreadsheetColumnID *new_column_id = spreadsheet_column_id_copy(src_column->id);
  SpreadsheetColumn *new_column = spreadsheet_column_new(new_column_id);

    new_column->display_name = BLI_strdup(src_column->display_name);
```

## GVArray (通用虚拟数组)

### 概念

GVArray 是类型擦除的数组容器，用于存储任意类型的数组数据。

```cpp
/* 来源: BLI_virtual_array.hh */
class GVArray {
private:
  const CPPType *type_;
  int64_t size_;
  /* 内部实现细节 */

public:
  /* 获取元素 */
  void get_to_uninitialized(int64_t index, void *r_value) const;

  /* 类型访问 */
  const CPPType &type() const { return *type_; }
  int64_t size() const { return size_; }

  /* 类型转换 */
  template<typename T>
  VArray<T> typed() const;
};
```

### 在 ColumnValues 中的使用

```cpp
/* 创建 ColumnValues */
GVArray data = VArray<float3>::from_span(positions);
ColumnValues values("Position", data);

/* 获取数据 */
const GVArray &array = values.data();
const CPPType &type = array.type();

/* 类型检查和使用 */
if (type.is<float3>()) {
  VArray<float3> typed = array.typed<float3>();
  float3 value = typed[0];
}
```

## 实际使用示例

### 从数据源创建列

```cpp
/* 来源: spreadsheet_data_source_geometry.cc (概念流程) */
void create_columns_from_data_source(DataSource &data_source, SpreadsheetTable &table)
{
  /* 1. 遍历所有列 */
  data_source.foreach_default_column_ids(
      [&](const SpreadsheetColumnID &id, bool is_extra) {
        /* 2. 创建列标识符 */
        SpreadsheetColumnID *new_id = spreadsheet_column_id_copy(&id);

        /* 3. 创建列 */
        SpreadsheetColumn *column = spreadsheet_column_new(new_id);

        /* 4. 获取列值以确定类型 */
        auto values = data_source.get_column_values(id);
        if (values) {
          /* 5. 分配运行时数据 */
          spreadsheet_column_assign_runtime_data(
              column,
              values->type(),
              values->name());

          /* 6. 计算初始宽度 */
          float width_px = values->fit_column_width_px(100);  /* 采样100个值 */
          column->width = width_px / SPREADSHEET_WIDTH_UNIT;
        }

        /* 7. 添加到表格 */
        spreadsheet_table_add_column(table, column);
      });
}
```

```cpp
/* 来源: spreadsheet_draw.cc (概念) */
void draw_column_header(const SpreadsheetColumn &column, const CellDrawParams ¶ms)
{
  const char *name = column.display_name ? column.display_name : column.id->name;

  /* 根据类型设置显示样式 */
  switch (column.data_type) {
    case SPREADSHEET_VALUE_TYPE_COLOR:
    case SPREADSHEET_VALUE_TYPE_BYTE_COLOR:
      /* 颜色列可能需要特殊显示 */
      draw_color_preview(params);
      break;

    case SPREADSHEET_VALUE_TYPE_INSTANCES:
      /* 实例引用显示图标 */
      draw_instance_icon(params);
      break;

    default:
      /* 普通文本 */
      draw_text(name, params);
      break;
  }
}
```

## 总结

列与值类型系统的关键特点：

1. **类型安全**: 完整的类型枚举和转换系统
2. **灵活**: 支持多种数据类型和显示格式
3. **高效**: 使用 GVArray 进行类型擦除，延迟格式化
4. **可扩展**: 易于添加新类型
5. **内存安全**: 严格的内存管理

**核心组件**:
- `SpreadsheetColumnID`: 列标识符
- `SpreadsheetColumn`: 列元数据
- `ColumnValues`: 列数据和值
- `eSpreadsheetColumnValueType`: 类型枚举
- `GVArray`: 类型擦除的数组容器

**类型支持**:
- 基本类型: bool, int8, int32, int64, float
- 向量: float2, float3, int2, int3
- 颜色: ColorGeometry4f, ColorGeometry4b
- 字符串: std::string
- 特殊: InstanceReference, Quaternion, float4x4, BundleItem
