#### 4. 为什么有 `intern` 文件夹
→ **阅读**: `004-intern_cycles目录架构详解.md`

**核心答案**:
- `intern` = 内部实现库
- 禁止 `source` → `intern` 反向依赖
- 保持 Cycles 独立性，便于维护
- 可独立编译成静态库

**目录对比**:
```
EEVEE: source/blender/gpu/shaders/*.glsl
Cycles: intern/cycles/
    ├── scene/          ← CPU 数据
    ├── kernel/         ← 渲染核心
    └── util/           ← 工具库
```
